# TeloPon (てろぽん) 🎙️✨

**AI가 방송자의 목소리를 듣고, 실시간으로 자막·실황·방송 지원을 수행하는 차세대 AI 어시스턴트**

![TeloPon 실행 화면](images/demo_ko.gif)

> 👉 **[TeloPon 최신 버전 다운로드](https://github.com/miyumiyu/TeloPon/releases/latest)**  
> ※ 초록색 "Code" → "Download ZIP"으로는 작동하지 않습니다. 반드시 Releases에서 다운로드하세요.

---

## 🌟 TeloPon으로 할 수 있는 것

### 💬 AI가 실시간으로 반응
방송자의 목소리를 Gemini Live API로 직접 처리. 음성 인식 → 텍스트 변환 → AI 전송 단계를 생략하여 **대화의 흐름을 끊지 않는 초저지연 응답**을 실현합니다.

### 📺 방송 플랫폼 연동
YouTube Live·Twitch·니코니코 생방송의 댓글을 AI가 자동으로 수집하여 반응. 방송의 썸네일·제목·설명도 AI에 공유되어, 방송 내용을 이해한 위에서 함께 방송을 성공적으로 이끕니다.

### 🎤 음성으로 모든 것을 조작
말을 걸기만 하면 AI가 방송을 지원. 설문 생성·제목 변경·클립 생성·OBS 녹화·썸네일 표시·트윗 게시까지, **음성만으로 조작**할 수 있습니다.

### 🎮 게임 화면을 보고 실황
OBS 연동으로 게임 화면이나 카메라 영상을 AI에게 보여주어, 시각 정보에 기반한 실시간 실황·리액션이 가능합니다. OBS 녹화 시작/정지도 음성으로 제어할 수 있습니다.

### 🐦 SNS 연동
X (Twitter)의 해시태그를 자동 수집하여 AI에 주입. 썸네일이나 게임 화면 첨부 트윗을 AI가 자동 게시합니다.

### 🔊 자막 읽기
Windows 기본 음성(SAPI5)·VOICEVOX·Gemini 3.1 Flash TTS로 자막을 자동 읽기. 방송에 AI의 "목소리"를 추가합니다.

### 📊 프레젠테이션 지원
PowerPoint 슬라이드쇼를 음성으로 조작. 슬라이드 넘기기/되돌리기·점프·블랙아웃. 슬라이드 노트를 AI에 자동 주입하여 프레젠테이션을 지원합니다.

### 🧠 나만의 AI 캐릭터
프롬프트(텍스트 파일)를 작성하기만 하면 어떤 캐릭터의 AI든 만들 수 있습니다. 츤데레, 사투리, 독설… 자유자재입니다.

### 🎨 자막 디자인 자유
CSS로 자막 외관을 완전히 커스터마이즈. 효과음도 설정할 수 있습니다.

## 🎤 AI 음성 명령 목록

방송 중에 말을 걸기만 하면 AI가 자동으로 조작합니다.

| 카테고리 | 음성 예시 | 동작 |
|---|---|---|
| **썸네일** | "썸네일 보여줘" | OBS에 썸네일 이미지를 표시 |
| | "썸네일 크게/작게/없애줘" | 확대·축소·숨기기 |
| **OBS 녹화** | "녹화 시작해" "녹화 멈춰" | OBS 녹화를 조작 |
| **UI 조작** | "설명 없애줘" "이름 없애줘" | 자막 창을 숨기기 |
| **YouTube** | "설문해줘" "제목 바꿔줘" | YouTube 측 조작 |
| **Twitch** | "클립해줘" "예측해줘" | Twitch 측 조작 |
| **니코니코** | "운영 댓글로 ○○라고 띄워줘" | 니코니코 측 조작 |
| **X (Twitter)** | "트윗해줘" "썸네일 붙여서 올려줘" | 트윗 게시 |
| **PowerPoint** | "다음 슬라이드" "프레젠 시작해" | 슬라이드 조작 |

> 각 명령은 해당 플러그인이 활성화된 경우에만 사용할 수 있습니다.

## 🔑 준비: 무료 API 키 받기

TeloPon을 사용하려면 Google Gemini API 키가 필요합니다 (**완전 무료·카드 등록 불필요**).

1. **[Google AI Studio](https://aistudio.google.com/)** 에 Google 계정으로 로그인
2. "**Get API key**" → "**API 키 생성**"
3. `AIza...`로 시작하는 문자열을 복사

👉 **[이미지로 보는 상세 설명](docs/ko/04_get_apikey.md)**

## 🛠️ 다운로드 및 실행 (Windows 전용)

1. **[Releases](https://github.com/miyumiyu/TeloPon/releases/latest)** 에서 최신 버전 ZIP을 다운로드
2. 우클릭 → "모두 압축 풀기"로 압축 해제
3. **TeloPon.exe**를 더블 클릭하여 실행

> ⚠️ ZIP을 압축 해제하지 않고 바로 더블 클릭하면 설정이 저장되지 않습니다.

## 🎛️ 화면(UI) 사용법

![UI 화면](images/ui_ko.png)

### ⚙️ AI 설정
| 항목 | 설명 |
|---|---|
| 🔑 **API Key** | Gemini API 키를 붙여넣고 "인증" |
| 🧠 **사용 모델** | 보통은 `Auto` 그대로 |
| 💭 **사고 레벨** | `minimal`(고속) ~ `high`(깊은 추론). 방송에는 `minimal` 추천 |
| 🧠 **AI 성격** | `prompts/` 폴더에서 대본을 선택 |
| 🎥 **방송자 이름** | AI가 당신을 부를 때 사용하는 이름 (필수) |

### 🎤 마이크 설정
사용할 마이크를 선택합니다. 레벨 미터가 **노란색**이 되는 음량이 최적입니다. 빨간색은 음이 깨집니다.  
**임계값 슬라이더**로 BGM이 인식되지 않는 경계선에 조정하는 것이 핵심입니다.

### ⏱️ 타임 설정
| 항목 | 설명 |
|---|---|
| **일반 / 특별** | 자막 표시 시간(초) |
| **발화 모드** | `None` / `자동 발화`(침묵 방지) / `자동 끼어들기`(정기 요약) |
| **개입 시간** | 자동 개입까지의 대기 시간(초) |

### 🔌 확장 기능 (플러그인)
오른쪽 패널에 플러그인이 나열됩니다. 각 플러그인의 "조작 패널"에서 설정·조작합니다.

### 🔧 플러그인 관리
플러그인 리스트 상단의 **"🔧 플러그인 관리"** 버튼으로 시작합니다.

| 탭 | 기능 |
|---|---|
| **활성** | UI에 표시 중인 플러그인을 관리. 불필요한 것을 숨기기 |
| **비활성** | 숨긴 플러그인을 다시 활성화 |
| **다운로드 가능** | 확장 플러그인을 원클릭으로 다운로드 & 즉시 사용 |
| **수동 추가** | 자작 플러그인의 `.py` 파일을 설치 |

### 🎨 테마·효과음
테마(CSS)로 자막 디자인을 선택합니다. 효과음 ON/OFF 및 음량 조절도 가능합니다.

### 🔗 OBS용 URL 복사
1. "🔗 OBS용 URL 복사"를 클릭
2. OBS Studio → "브라우저" 소스 추가 → URL을 붙여넣기
3. 크기 **너비 1920 × 높이 1080**으로 설정

👉 **[OBS 자막 조작 가이드 (이미지 포함)](docs/ko/05_obs_control.md)**

**👉 모든 설정이 완료되면 "🔴 라이브 연결 시작"!**

## 🚥 상태 표시 보는 법

마이크 설정 아래의 "상태" 표시는 AI의 현재 상태를 실시간으로 보여줍니다.

| 표시 | 의미 |
|---|---|
| ⬛ **대기 중** | 라이브 연결 전 |
| 🟢 **방송 중!** | 음성을 수집할 수 있는 상태 |
| 🎧 **듣는 중...** | 당신의 목소리를 감지 중 |
| 🧠 **생각 중...** | 응답을 생각하는 중 |
| 🗣️ **출력 중...** | 자막을 출력하는 중 |
| 👻 **발화 스킵** | AI가 "조용히 있자"고 판단 (정상) |
| 👻 **노이즈 스킵** | 기침·잡음·처리 지연 |
| ⚠️ **인터럽트 감지** | 당신이 AI의 발화를 끊음 |
| 🚫 **안전 제한** | AI의 안전 필터가 작동 |

## 💡 장시간 방송 팁

약 30분이 지나면 AI의 기억이 쌓여 처리가 무거워질 수 있습니다.  
화제가 바뀌는 타이밍에 **"⬛ 연결 끊기" → "🔴 라이브 연결 시작"**을 누르기만 하면 AI의 기억이 초기화되어 빠른 응답으로 돌아옵니다.

## ⚠️ 얼리 액세스 버전에 대하여

1. **시작 시 확인**: 온라인으로 버전 확인을 수행합니다. 오래된 버전은 사용이 중지될 수 있습니다.
2. **무보증**: 이 도구는 "있는 그대로" 제공됩니다. 사용은 자기 책임으로 부탁드립니다.
3. **분석·재배포 금지**: 실행 파일의 리버스 엔지니어링·수정·무단 재배포를 금지합니다 (플러그인·프롬프트의 작성·배포는 자유입니다).
