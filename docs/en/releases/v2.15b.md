# 🚀 Release V2.15b: AI Icon, UI Redesign & Stability Improvements!

> 👉 **[Download TeloPon v2.15b](https://github.com/miyumiyu/TeloPon/releases/download/v2.15b/TeloPon-215b.zip)**

This update adds AI icon display on telops, reorganizes the settings UI, and improves connection stability.

---

## ✨ AI Icon Feature

### 🆕 Display AI Icon on Telops

You can now display a custom icon image on the left side of telops to match your AI's personality.

* Set an image via the "AI Icon" button in the UI
* **Auto-resizes to match telop height** (square, center-cropped)
* No icon = same display as before
* Supported: variety / news / digital / cute / simple
* Not shown on reply / explain / nameplate

---

## ✨ UI Redesign

The settings layout has been reorganized for better clarity.

* **⚙️ AI Settings**: API Key, auth status, model, thinking level, AI personality, AI icon
* **📡 Streaming Settings**: Streamer name, microphone, level meter, threshold, theme

---

## 🔧 Stability Improvements

### Auto-Recovery When AI Becomes Unresponsive

Added an automatic session reconnection mechanism when the AI stops responding for an extended period.
This significantly reduces the occurrence of the AI going silent during a stream.

---

## 🔧 Other Fixes

* Fixed an issue where some plugins stopped working in v2.14b (added ctypes.wintypes to build)
* Fixed telop display delay carrying over from the previous session on startup (now always resets to 0)
