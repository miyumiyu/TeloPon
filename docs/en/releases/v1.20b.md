# 🌐 Release V1.20b: 4-Language Support & Twitch Plugin Bundled!

This update focuses on **going global** and **expanding streaming platform support**.
The UI and documentation now support 4 languages, making TeloPon more accessible to streamers around the world.
Plus, a new Twitch chat and stream integration plugin is now bundled as a standard plugin — so you can start streaming on Twitch right away, not just YouTube!

---

## ✨ What's New

### 🌐 UI Available in 4 Languages (日本語 / English / 한국어 / Русский)

TeloPon's UI (buttons, status, menus, etc.) now automatically switches based on your OS language setting.

* Supports **Japanese**, **English**, **Korean**, and **Russian**.
* The OS language is detected at startup; unsupported languages fall back to English.
* Plugin UIs are also fully localized.

---

### 🎮 Twitch Live Plugin Now Bundled (TwitchPlugin.py)

A new plugin supporting Twitch streaming is now included as a **standard bundled plugin** — no separate installation required.

* **Chat reading**: Just enter a channel name to receive viewer comments in real time and pass them to the AI. **Works without a Client ID** (via anonymous IRC).
* **Stream info injection**: Set up a Client ID / Client Secret via the Twitch Developer Console to also send the stream's "title", "category", and "thumbnail" to the AI automatically.
* **Accepts both URL and channel name**: Both `https://twitch.tv/mychannel` and `mychannel` are recognized.

👉 [Twitch Live Plugin — Detailed Guide](../plugins/TwitchPlugin.md)

---

## 💡 Tips for Streamers

**Streaming on both Twitch and YouTube at the same time?**
The Twitch plugin and YouTube plugin can be enabled simultaneously. Feed comments from both platforms to the AI at once for a truly all-inclusive streaming experience where no viewer is left out!

**Want to try the Twitch plugin right now?**
You can skip the Client ID setup entirely. Just enter a channel name, press "Connect", then press "🔴 Start Live Connection" — Twitch chat integration works immediately.
