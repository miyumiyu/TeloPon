# 🚀 Release V2.10b: Mahjong Tile Display, Viewer Avatars & Stability Improvements!

> 👉 **[Download TeloPon v2.10b](https://github.com/miyumiyu/TeloPon/releases/download/v2.10b/TeloPon-210b.zip)**

This update focuses on **further stabilization and optimization for Gemini 3.1 Flash Live**, along with mahjong streaming support, viewer avatar display, and more.

---

## ✨ What's New & Improved

### ⚡ Gemini 3.1 Flash Live Optimization & Stability

The connection handling for Gemini 3.1 Flash Live has been fully refactored.

* Telop response speed has improved further, enabling near real-time display
* Session reconnection stability improved

---

### 🆕 Enhanced Comment Reply (window-reply)

The comment reply telop `window-reply` has been significantly upgraded in design and functionality.

* Changed to a **speech bubble style**, placed in the **top-left** of the screen
* Displays commenter **avatars (thumbnails)** from **YouTube and Twitch** in reply telops
  * YouTube avatars are fetched automatically when comments are received
  * Twitch requires Helix API credentials (Client ID & Secret ID) → [Twitch Plugin Settings](../plugins/TwitchPlugin.md)

---

### 🆕 Mahjong Streaming Support

New features have been added for mahjong streaming.

#### Mahjong Tile Inline Display Tags

Writing tile tags in a prompt will automatically display tile images in the telop.

| Type | Tags |
|---|---|
| Characters (Man) | `<M1>` 〜 `<M9>` |
| Bamboo (Sou) | `<S1>` 〜 `<S9>` |
| Circles (Pin) | `<P1>` 〜 `<P9>` |
| Honors | `<J1>`East `<J2>`South `<J3>`West `<J4>`North `<J5>`White `<J6>`Green `<J7>`Red |

#### Mahjong Advisor Prompt `r_mahjong.txt`

* Combined with OBS Capture, the AI can display discard advice telops for your hand
* Designed to give advice based only on tiles visibly confirmed in the screen capture
* ⚠️ **The AI may misread tiles in the image and mention tiles that don't exist.** Please enjoy it as entertainment!

---

### 🔧 Stability & Other Improvements

* Prevented telop misfires caused by the AI misidentifying silence after generating output
* Other minor bug fixes
