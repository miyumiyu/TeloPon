# 🚀 Release V2.14b: Plugin Decoupling, Telop Visibility Improvements & VOICEVOX TTS!

> 👉 **[Download TeloPon v2.14b](https://github.com/miyumiyu/TeloPon/releases/download/v2.14b/TeloPon-214b.zip)**

This update introduces a decoupled plugin architecture, improved telop visibility, and a new VOICEVOX text-to-speech extension plugin.

---

## ✨ Decoupled Plugin Architecture

### 🆕 Fire-and-Forget Design

The telop output hook (`on_telop_output`) has been redesigned with a **fire-and-forget** approach.

* Core **never waits** for plugin processing to complete
* Errors or delays inside plugins have **no impact on the system**
* Delay control is separated via `set_telop_delay()` / `get_telop_delay()`

This provides a safe foundation for third-party plugins to operate reliably.

👉 **[Plugin Development Guide (details)](https://github.com/miyumiyu/TeloPon/blob/main/docs/en/03_plugin_dev.md)**

## ✨ Telop Visibility Improvements

Stack opacity for normal telops has been adjusted to improve visibility of older telops.

| Position (newest → oldest) | Before | After |
|---|---|---|
| 1st (newest) | 100% | 100% |
| 2nd | 50% | **75%** |
| 3rd | 30% | **50%** |
| 4th and beyond | 10% | **25%** |

---

## ✨ VOICEVOX TTS Extension Plugin

### 🆕 VOICEVOX Text-to-Speech Plugin

A new extension plugin that reads aloud AI-generated telop text using the **VOICEVOX** speech synthesis engine in real time.

* **Character & Style selection**: All VOICEVOX characters + styles (Normal/Sweet/Tsun-tsun, etc.) via dropdown
* **Icon display**: Character/style icon shown in the settings panel
* **5 audio parameters**: Speed, volume, pitch, intonation, and pre-silence with real-time adjustment
* **TOPIC readout**: Optionally reads the top line text along with the main text
* **Telop display control**: TTS-only mode (telop hidden) or delayed display
* **Auto connection check**: Automatically verifies VOICEVOX connection; UI grays out if unavailable

👉 **[VOICEVOX TTS Plugin Guide](https://github.com/miyumiyu/TeloPon/blob/main/docs/en/plugins/voicevox_plugin.md)**

> 💡 Requires separate installation of [VOICEVOX](https://voicevox.hiroshiba.jp/).
