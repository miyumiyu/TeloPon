# 🚀 Release V2.12b: Reply Card Redesign & Explain Position Memory!

> 👉 **[Download TeloPon v2.12b](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

This update redesigns `window-reply` as an SNS-style reply card with avatar display, slide animation, and glow effects. The explain window (`window-explain`) now persists its position and zoom scale across OBS restarts.

---

## ✨ New Features & Improvements

### 🆕 window-reply SNS Card Design

`window-reply` telops have been redesigned to look like SNS reply cards.

* **Circular avatar icon** for the commenter (supports YouTube profile images, etc.)
* **↩ reply badge** displayed at the bottom-right of the avatar
* **Slide-in animation** on telop appearance (direction auto-detected from anchor position)
* Dark background with **glow effect** (all 4 themes supported: default / comic / honwaka / tsundere)

---

### 🆕 Explain Position & Scale Persistence

The position and zoom level of `window-explain` are now saved to `localStorage`.

* Dragged position is **maintained after OBS restarts**
* Wheel zoom scale is **also preserved**
* **Double-click to reset** position and scale

---

### 🔧 Other Fixes

* Fixed saved zoom scale not being applied to `window-reply` after slide animation
* Fixed telop briefly jumping to top-left corner when double-clicking to reset
* Fixed OBS browser source aggressively caching CSS (server-side no-cache headers + URL timestamp)
