# 🚀 Release V2.11b: Telop Stacking, Zoom & Anchor Positioning!

> 👉 **[Download TeloPon v2.11b](https://github.com/miyumiyu/TeloPon/releases/download/v2.11b/TeloPon-211b.zip)**

This update significantly enhances the telop display UI on OBS, introducing vertical stacking, zoom control, and drag-to-anchor positioning.
(🙏 Special Thanks **[@mo9mo9ke6ri](https://x.com/mo9mo9ke6ri)**)

---

## ✨ New Features & Improvements

### 🆕 Telop Vertical Stacking

Multiple telops now stack vertically without overlapping.

* Telops are split into two independent groups: **`window-reply` group** (left side) and **normal group** (right side)
* Each group displays up to **4 telops** simultaneously
* New telops appear at the top (or bottom), while older ones slide down
* Older telops gradually fade out (100% → 50% → 30% → 10%)

---

### 🆕 Anchor Positioning (Drag to Set Appearance Position)

Dragging a telop sets the anchor point — all subsequent telops in the same group will appear from that position.

* **Anchor position is saved to the browser's `localStorage`** and persists after OBS restarts
* If the anchor Y coordinate is below 3/4 of the screen height, the stacking direction **automatically switches to upward**

---

### 🆕 Wheel Zoom to Resize Telops

Hover over a telop and use the mouse wheel to resize it.

* Zoom level is saved independently for the **`window-reply` group** and the **normal group**
* The saved size is applied when the next telop in the same group appears
* **Double-click to reset** size (and position)
* Requires **"Interactive"** to be enabled in the OBS browser source settings
