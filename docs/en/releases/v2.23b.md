## 🆕 New Features

### 🕐 Accurate Time Responses (Function Calling)
Ask "What time is it?" or "What's today's date?" and the AI will retrieve the system clock and respond with the accurate local time.  
Powered by Gemini Live API Function Calling — no more guessing or hallucinating the time.

### 📺 Thumbnail & Description Injection ON/OFF
Added checkboxes to YouTube Live, YouTube Live+, Twitch, and Niconico plugins to toggle whether thumbnail images and stream descriptions are sent to the AI at stream start.  
Turn them OFF to save prompt tokens when not needed.

### 💬 Post Telops to Twitch Chat
Added a "Post telops to chat" checkbox in the Twitch plugin feature settings.  
When enabled, AI-generated telop text is automatically posted to Twitch chat.
