# 🚀 Release V2.13b: Plugin Telop Output Hook & TelopViewer Plugin!

> 👉 **[Download TeloPon v2.13b](https://github.com/miyumiyu/TeloPon/releases/download/v2.13b/TeloPon-213b.zip)**

This update adds a telop output hook to the plugin framework, allowing plugins to receive AI output in real time. The built-in telop monitor has been replaced with a plugin-based implementation.

---

## ✨ Telop Output Hook for Plugins

### 🆕 on_telop_output Hook

A new `on_telop_output` hook lets plugins receive data every time the AI outputs a telop.

* Plugins receive the full data: **topic / main / window / layout / badge**
* Tags like `<b1>` `<b2>` and mahjong tiles (`<P1>`, etc.) are passed as raw data
* Return value controls OBS telop display

| Return Value | Behavior |
|---|---|
| `0` | Instant display (default) |
| `1+` | Display after N seconds (delay) |
| `-1` | Suppress OBS display (plugin still receives data) |

### 🛡️ Safety Mechanisms

Protection against plugin issues affecting the entire system:

* Each plugin's `on_telop_output` runs with a **1-second timeout**
* Plugins that timeout **3 times** are automatically excluded from future hook calls

👉 **[Plugin Development Guide (on_telop_output details)](https://github.com/miyumiyu/TeloPon/blob/main/docs/en/03_plugin_dev.md)**

---

## ✨ Telop Monitor Replaced by TelopViewer Plugin

### 🆕 TelopViewer Plugin

The built-in "Telop Monitor" has been removed and rebuilt as the **TelopViewer plugin** using the `on_telop_output` hook.

* **Real-time display** of AI telop output in the control panel
* **B1/B2 tag color coding** (B1=red, B2=cyan)
* **Display delay configurable via dropdown** (0/1/2/3/5/10/-1)
* Simply open from the "Control Panel" button in the plugin list
