## ✨ What's New & Improved

### 🆕 Gemini 3.1 Flash Live Support

TeloPon now supports the latest **Gemini 3.1 Flash Live** model.

* Auto mode automatically selects Gemini 3.1 Flash Live
* Delivers more stable connections and lower-latency responses

---

### 🆕 New Feature: Telop Monitor

A new monitor window has been added, displaying a chronological history of all AI-generated telops directly within the TeloPon UI.

* Toggle on/off with the "Telop Monitor OFF/ON" button below the OBS copy button
* No need to check OBS during a live session — telop history is always visible in the app
* Enabling the monitor after launch will also display all previous telops logged in the current session
