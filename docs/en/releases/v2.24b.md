## 🆕 New Features

### ⏰ Timer & Alarm (Function Calling)
The AI can now set timers and alarms directly from your voice — and notify you automatically when they fire.

- **"Set a 5-minute timer"** → telop notification after 5 minutes
- **"Let me know at 5 PM"** → telop notification at 17:00
- **"Cancel the timer"** → cancels the active timer

And it also handles **action-linked requests**:

- **"End the poll in 3 minutes"** → the AI automatically issues the poll-end command after 3 minutes
- **"Change the title in 5 minutes"** → the AI automatically issues the title-change command after 5 minutes

Both relative durations (seconds/minutes/hours) and absolute times (HH:MM) are supported. You can delegate scheduled stream operations directly to the AI.

### 🔄 Live Reload
A new **Reload** button in live mode lets you wipe the AI's context (memory) without interrupting the stream.

- **No standby telop** — the stream keeps flowing while the AI session resets
- Plugins, mic monitor, and timers keep running
- Ideal for recovering when the AI starts carrying forward mistaken information or misrecognitions

> **Response to issue [#13](https://github.com/miyumiyu/TeloPon/issues/13) (mic mute request)**: rather than manual muting (which inevitably misses timing), we went with removing contaminated context after the fact.

---

## 🧠 AI Response Improvements

### Suppressed Time-Fillers
Fixed an issue where the AI would spontaneously announce the current time as filler ("The time is now X:XX") when there was nothing else to say. The AI now only reports the time when actually asked, thanks to strengthened system prompts (all 4 languages).

## 🔌 Extension Plugins

### 🔊 Gemini TTS Plugin (New)
A brand-new high-quality telop TTS plugin powered by the latest **Gemini 3.1 Flash TTS Preview**.

- 30 preset voices
- Expressive control via badge (emotion) / character instructions / speed / delay
- Uses TeloPon's existing Gemini API key — no extra authentication
- Output device selection

> ⚠️ **Note**: Gemini TTS API has strict rate limits. On the free tier, you'll hit the limit after about 5 calls. Please use this plugin as a **trial/demo** experience.
