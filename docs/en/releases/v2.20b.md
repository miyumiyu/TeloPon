## 🆕 New Features

### 📺 Full Streaming Platform API Support
- **YouTube Live**: Read/write comments, create/tally polls, change stream title, get viewer count, Super Chat support (OAuth)
- **Twitch**: Read/write comments, polls, predictions (betting), create clips, change title, get viewer count (OAuth2 Device Code Flow)
- **Niconico Live**: Read comments, gift/ad notifications, create/tally polls, post operator comments, get viewer count (promoted to bundled plugin)

### 🔧 Plugin Manager
- Enable/disable plugins from the UI
- Download and use extension plugins with one click
- Manual installation of custom plugins also supported

### 🎤 Thumbnail Voice Control
- Say "show thumbnail," "make it bigger," or "hide it" to display/control thumbnail images in OBS
- Automatically enabled when connected to a streaming platform

### 🎮 Enhanced OBS Integration
- Voice control for OBS recording ("start recording," "stop recording")
- Select capture targets from a dropdown list of scenes

### 🗣️ System Voice Commands
- Say "close the description" or "hide the nameplate" to have the AI dismiss on-screen windows

## 🔄 Improvements & Changes

- Unified plugin naming convention (📺 Streaming / 🐦 SNS / 💬 Chat / 🔊 TTS / 🎮 Screen / 📊 Presentation)
- Added stream info sharing mechanism (stream URL and thumbnails shared across plugins)
- Fixed UTF-8 encoding issues in the VCI OSC plugin
- Simplified the cue card and image injection UI
- Fixed flickering of mahjong tile images
- Improved AI recognition of regional time zones (Japan: JST / Korea: KST / Russia: MSK)
