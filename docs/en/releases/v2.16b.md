# 🚀 Release V2.16b: Major Twitch Upgrade, YouTube OAuth, Command Permissions!

> 👉 **[Download TeloPon v2.16b](https://github.com/miyumiyu/TeloPon/releases/download/v2.16b/TeloPon-216b.zip)**

This update brings a major Twitch plugin upgrade, a new YouTube OAuth integration plugin, and command permission management.

---

## 🎮 Major Twitch Plugin Upgrade

With OAuth2 authentication support, the Twitch integration has been completely revamped from read-only to full-featured.

- **Write comments** - Say "write 'hello' in chat" and the AI posts to Twitch chat
- **Polls** - Say "create a poll about favorite colors" to start a vote, "tally the results" to announce
- **Predictions (Betting)** - Say "let viewers bet on whether I can beat this boss" for channel point betting (Affiliate/Partner)
- **Clip creation** - Say "that was great! clip it!" to save the last 30 seconds
- **Title change** - Say "change the title to chill stream" for instant updates
- **Viewer count** - Say "how many viewers?" and the AI tells you
- **Bot exclusion** - Double-click to add nightbot etc. to the exclusion list
- **Auth Mode / IRC Mode** - Switch between full features or chat-read-only (no auth required)

---

## ▶️ YouTube OAuth Plugin Added to Extensions

Released an advanced YouTube integration plugin using YouTube Data API + OAuth2.

- **Write comments** - Say "post a comment saying hello" and the AI posts to YouTube chat
- **Polls** - Say "create a poll" to make a YouTube poll, "close the poll" for results
- **Title change** - Say "change the title to gaming stream" to update the stream title
- **Viewer count** - Say "how many viewers?" to get the concurrent viewer count
- **Broadcast selection** - Choose from a list of scheduled broadcasts or specify by URL
- **Auto permission detection** - Automatically detects owner/manager status and shows available features

A GCP setup guide with screenshots is also available.

👉 [TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)

---

## 🛡️ Command Permissions Added to Twitch / YouTube OAuth

Added "Viewer" checkboxes to each feature in Twitch and YouTube OAuth plugins.  
Control whether viewer comments can trigger commands on a per-feature basis.  
For example, title changes and polls can be restricted to streamer voice commands only, while viewer comments are ignored.

---

## 🎤 Close Explain/Nameplate Windows by Voice

Say "close the explanation" or "hide the nameplate" and the AI closes the on-screen window.
