# 🎮 Twitch連携ツール (TwitchPlugin.py)

このプラグインは、TwitchチャンネルのURLまたはチャンネル名を指定するだけで、**視聴者のチャットコメントをリアルタイムに拾ってAIに反応させる**ことができる拡張機能です。

さらに、**TwitchのClient IDとClient Secret**（任意）を設定すれば、配信の「タイトル」「カテゴリ」「サムネイル画像」も自動取得してAIに送り込むことができます。**Client IDなしでも、チャット読み取りのみで動作します。**

---

## ⚙️ 使い方

### 1. 設定画面を開く

TeloPonのメイン画面右側、「拡張機能（プラグイン）」パネルにある **「Twitch連携ツール」** の **「⚙️ 設定」** ボタンをクリックします。

### 2. チャンネル名を入力する

上部の入力欄に、配信のTwitchチャンネル名（例：`mychannel`）またはURL（例：`https://twitch.tv/mychannel`）を入力します。

### 3. Client ID / Client Secret を設定する（任意）

ストリームタイトル・カテゴリ・サムネイルもAIに共有したい場合は、**「🔑 Client ID」** と **「🔒 Client Secret」** を入力してください。

> ⚠️ **未入力のままでもチャット読み取りは動作します。** チャット連携だけ使いたい場合はこの手順を省略できます。

**Client ID / Client Secretの取得方法：**

1. [https://dev.twitch.tv/console](https://dev.twitch.tv/console) にTwitchアカウントでログインします。
2. 「アプリケーションを登録」でアプリを新規作成します（名前・カテゴリは任意）。
3. 作成後に表示される「Client ID」と「新しいシークレット」をコピーして入力します。

### 4. 「接続」ボタンを押す

入力が完了したら **「接続」** ボタンを押してください。

* **Client IDなしの場合**：「✅ チャット接続成功」と表示され、チャット読み取りが有効になります。
* **Client IDありの場合**：ストリーム情報とサムネイルが取得・表示され、「✅ 接続成功」と表示されます。

バッジが緑色の `ON` に変わったことを確認したら「閉じる」で画面を閉じてください。

### 5. ライブ接続を開始する

TeloPonのメイン画面に戻り、**「🔴 ライブ接続開始」** ボタンを押してAIと通話を開始します。

これで連携は完了です！AIとの通話が始まると同時に、Twitchの情報が共有され、視聴者のコメントにAIが自動で反応するようになります。

---

## 🧠 AIに連携される情報

| 情報 | 条件 | タイミング |
|---|---|---|
| 配信タイトル・カテゴリ | Client ID設定時のみ | ライブ接続開始時 |
| サムネイル画像 | Client ID設定時のみ | ライブ接続開始から約5秒後 |
| 視聴者のチャットコメント | 常に（匿名IRC経由） | リアルタイム（5秒ごとにまとめて送信） |

### コメントの注入動作

* チャットコメントは**5秒ごと**、または**10件溜まった時点**でまとめてAIに送信されます。
* 各コメントは `[COMMENT] ユーザー名: コメント内容` の形式でAIに届きます。

---

## ⚠️ ご利用上の注意点

* **接続の順番について**
  ストリームタイトル・カテゴリをAIに確実に認識させるため、**「🔴 ライブ接続開始」ボタンを押す前に、このツールで「接続」しておく**ことを強くおすすめします。（通話途中で接続した場合、タイトル・カテゴリは認識されません）

* **オフライン中のチャンネルについて**
  Client IDありで接続する場合、対象チャンネルがオフライン（配信中でない）のときはストリーム情報が取得できません。チャット接続は引き続き動作します。

* **コメントが多すぎる場合**
  視聴者が多く、コメントが大量に流れる配信の場合、AIがすべてのコメントに反応しようとして喋りっぱなしになる可能性があります。
