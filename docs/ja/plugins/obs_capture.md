# 🎮 OBS連携 (obs_capture.py)

OBS Studioの画面をキャプチャしてAIに見せたり、OBSの録画操作を音声で制御できるプラグインです。

ゲーム画面やカメラ映像をAIに見せることで、**視覚情報に基づいたリアルタイム実況やリアクション**が可能になります。

---

## 🛠️ 準備：OBS側の設定 (OBS WebSocket)

TeloPonからOBSを操作するために、OBS側の「WebSocket」機能を有効にする必要があります。

1. **OBS Studio** を起動
2. メニューの **「ツール」** → **「WebSocket サーバー設定」** をクリック
3. **「WebSocketサーバーを有効にする」** にチェック
4. **サーバーポート** はデフォルトの `4455` でOK
5. **認証を有効にする** にチェックし、パスワードを設定
6. 「適用」→「OK」

---

## ⚙️ TeloPon側の設定

### 1. 操作パネルを開く

TeloPonのメイン画面 → 「拡張機能」→ **「🎮 OBS連携」** の **「🖥️ 操作パネル」** をクリック。

### 2. OBS WebSocket 接続設定

| 項目 | 説明 |
|---|---|
| **ホスト / ポート** | 通常は `127.0.0.1` : `4455`（同じPCの場合） |
| **パスワード** | OBS側で設定したパスワード |

### 3. キャプチャー設定

| 項目 | 説明 |
|---|---|
| **配信中、一定間隔で自動キャプチャしてAIに送る** | ONにすると指定間隔で自動送信 |
| **自動送信の間隔（秒）** | 30〜60秒がおすすめ（短すぎるとAIが喋りっぱなしに） |
| **対象シーン** | プルダウンからOBSのシーンを選択（クリックで自動取得） |
| **テストキャプチャ** | シーン選択の横のボタンで接続テスト＆プレビュー表示 |
| **指示テキスト** | AIに画像と一緒に送るカンペ |
| **今すぐ画面をAIに送る** | 決定的瞬間に手動で送信 |

### 4. 録画操作

| 項目 | 説明 |
|---|---|
| **配信者の音声で録画を操作する** | ONにすると音声で録画開始/停止が可能 |
| **状態表示** | 現在の録画状態（録画中 / 停止中）が自動表示 |

---

## 🎤 音声コマンド

録画操作のチェックがONの場合、以下の音声コマンドが使えます。

| 音声例 | 動作 |
|---|---|
| 「録画開始して」「録画して」 | OBSの録画を開始 |
| 「録画停止して」「録画止めて」 | OBSの録画を停止 |

---

## ⚡ 配信中の活用方法

### 自動送信
キャプチャのチェックをONにしておけば、設定した間隔で自動的に画面がAIに送られ続けます。

### 手動送信
「今ここを見てほしい！」という瞬間（ガチャ結果、ボス撃破など）に、**「⚡ 今すぐ画面をAIに送る」** ボタンで即座に送信できます。

---

## ⚠️ ご利用上の注意

- **画面の解像度**: AIへ送る画像は自動縮小されます。画面の隅の小さな文字は読めない場合があります。
- **シーン名**: プルダウンから選択するため、入力ミスの心配はありません。OBSが起動していない場合はリストが空になります。
- **録画操作**: OBSが録画対応の設定になっていない場合はエラーになります。
