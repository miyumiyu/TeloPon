# 🎮 Twitch連携ツール v2 (TwitchPlugin.py)

Twitch Helix API + OAuth2 Device Code Flow による高機能 Twitch 連携プラグインです。  
配信者の音声指示でAIがTwitchの各種操作を自動実行します。

---

## 🌟 主な機能

| 機能 | 説明 | アカウント制限 |
|---|---|---|
| チャットコメント読み取り | 視聴者のコメントをAIに注入 | 全ユーザー（認証不要） |
| チャットコメント書き込み | AIがTwitchチャットに投稿 | 全ユーザー |
| アンケート（Poll）作成・締切 | 視聴者投票を作成・集計 | **アフィリエイト/パートナーのみ** |
| 予測（Prediction）作成・確定 | 視聴者がポイントを賭ける | **アフィリエイト/パートナーのみ** |
| 配信タイトル変更 | 音声でタイトルを変更 | 全ユーザー |
| クリップ作成 | 直前の約30秒をクリップ | 全ユーザー |
| 視聴者数取得 | 現在の同時視聴者数を取得 | 全ユーザー |
| Bot除外 | 特定ユーザーのコメントをAI注入から除外 | - |

## ⚙️ 連携までの流れ

### Step 1: Twitch Developer Console でアプリ登録

1. [Twitch Developer Console](https://dev.twitch.tv/console/apps) にTwitchアカウントでログイン
2. 「アプリケーションを登録」をクリック
3. 入力内容:

| 項目 | 入力例 | 説明 |
|---|---|---|
| 名前 | `TeloPon` | 任意の名前 |
| OAuthリダイレクトURL | `https://localhost` | 使用しないがフォーム上必須 |
| カテゴリ | `Application Integration` | 任意 |

4. 作成後、アプリの詳細画面で:
   - **Client ID** をコピー
   - **「新しいシークレット」** ボタンで **Client Secret** を生成してコピー

### Step 2: TeloPonで認証

1. TeloPonのメイン画面 → 「拡張機能」→「Twitch連携ツール」の歯車をクリック
2. 設定画面で:
   - **Client ID** を貼り付け
   - **Client Secret** を貼り付け
   - **「Twitchで認証する」** ボタンをクリック
3. ブラウザが開き、認証用のURLが表示されます
   - URLの横の **「コピー」** ボタンでクリップボードにコピーできます
   - ブラウザでTwitchにログインし、認証を許可します
4. 認証成功すると、ユーザー名とアカウントタイプが表示されます

```
🖼️ miyumiyuna5  [一般]
```

> 認証中に中止したい場合は **「認証キャンセル」** ボタンを押してください。

### Step 3: 接続モードを選択

設定画面には2つの接続モードがあります。

#### 認証モード（自分の配信・全機能）

自分のチャンネルに接続し、全機能が使えます。

- **「接続」** ボタンを押すと接続開始
- 配信情報（タイトル・カテゴリ・サムネイル）が表示されます
- 配信前でもタイトル・カテゴリは表示されます

#### IRCモード（誰の配信でも・コメント読取のみ）

任意のチャンネルのコメントを読み取ります。認証不要。

- チャンネル名またはURLを入力して **「接続」** ボタン
- コメント読み取りのみ（書き込みや操作系は不可）

### Step 4: ライブ配信を開始

TeloPonのメイン画面で **「🔴 ライブ接続開始」** を押します。  
AIとの通話が始まると同時に、Twitchとの連携が有効になります。

## 🎙️ 発話例（配信者の音声コマンド）

### コメント書き込み

配信者がAIにTwitchチャットへの書き込みを指示します。

| 発話例 | AIの動作 |
|---|---|
| 「チャットに"こんにちは"って書いて」 | `[CMD]TW:comment こんにちは` → チャットに投稿 |
| 「"配信始まるよ"ってコメントして」 | `[CMD]TW:comment 配信始まるよ` → チャットに投稿 |

### アンケート（Poll）※アフィリエイト/パートナーのみ

| 発話例 | AIの動作 |
|---|---|
| 「好きな色をアンケートして」 | `[CMD]TW:poll create 好きな色は？\|赤\|青\|緑 60` → 60秒のアンケート作成 |
| 「次に何のゲームやるかアンケート取って」 | `[CMD]TW:poll create 次のゲームは？\|APEX\|マイクラ\|スプラ 120` → 120秒のアンケート |
| 「集計して」「アンケート締め切って」 | `[CMD]TW:poll close` → 締切＆結果をAIにフィードバック |

> AIが結果を受け取り、テロップで視聴者に発表します。

### 予測（Prediction）※アフィリエイト/パートナーのみ

視聴者がチャンネルポイントを賭けるベッティング機能です。

| 発話例 | AIの動作 |
|---|---|
| 「このボス倒せるか賭けさせて」 | `[CMD]TW:predict create このボスを倒せるか？\|倒せる\|倒せない 120` |
| 「予測して。勝つか負けるか」 | `[CMD]TW:predict create 次の試合の結果は？\|勝ち\|負け 180` |
| 「正解は1番！」 | `[CMD]TW:predict resolve 1` → 選択肢1で確定、ポイント分配 |
| 「予測キャンセルして」 | `[CMD]TW:predict cancel` → 全員にポイント返還 |

### タイトル変更

| 発話例 | AIの動作 |
|---|---|
| 「タイトルを"雑談配信"に変えて」 | `[CMD]TW:title 雑談配信` → 配信タイトルを変更 |
| 「タイトル変えて、マイクラ建築回にして」 | `[CMD]TW:title マイクラ建築回` |

### クリップ作成

配信の直前約30秒を短い動画として自動保存します。

| 発話例 | AIの動作 |
|---|---|
| 「今の面白かった！クリップして」 | `[CMD]TW:clip` → クリップ作成 |
| 「今の切り抜いて」 | `[CMD]TW:clip` |

> 配信中のみ有効です。

### 視聴者数取得

| 発話例 | AIの動作 |
|---|---|
| 「今何人見てる？」 | `[CMD]TW:viewers` → 視聴者数をAIにフィードバック |
| 「視聴者数教えて」 | `[CMD]TW:viewers` |

> AIが結果を受け取り、テロップで配信者に伝えます。

## 🛡️ コマンドの視聴者許可設定

各機能には **「視聴者可」** チェックボックスがあります。  
これにより、視聴者のコメントからコマンドを実行できるかどうかを制御できます。

| 設定 | 動作 |
|---|---|
| ☑ 視聴者可 ON | 視聴者が「アンケートして」等とコメントしたらAIが実行 |
| ☐ 視聴者可 OFF | 配信者の音声指示のみで実行。視聴者コメントからは実行しない |

### デフォルト設定

| 機能 | デフォルト |
|---|---|
| コメント書き込み | ☑ ON |
| アンケート | ☑ ON |
| 予測 | ☑ ON |
| タイトル変更 | ☑ ON |
| クリップ | ☑ ON |

> **注意:** タイトル変更やクリップなど、影響が大きい操作を視聴者に許可するかは慎重に判断してください。

## 👤 ユーザー管理（Bot除外）

設定画面の右側にユーザー管理パネルがあります。

### 最近のユーザー

チャットでコメントしたユーザーが最大10件表示されます。  
**ダブルクリック** で除外リストに移動します。

### 除外リスト

除外リストに入ったユーザーのコメントはAIに注入されません。  
**ダブルクリック** で除外を解除できます。

- 認証ユーザー自身はデフォルトで除外リストに追加されます
- `nightbot`、`streamelements` 等のBotを除外するのに便利です
- 除外リストは設定ファイルに永続化されます

## 🔄 ライブ中の後接続

ライブ配信を開始した後からTwitch接続した場合でも、  
AIにコマンドプロンプトがメッセージとして自動注入されます。

> ただし、ライブ開始前に接続しておくことを推奨します。  
> ライブ開始前に接続した場合は `system_instruction` に含まれるため、より確実です。

## ⚠️ ご利用上の注意点

### トークンの有効期限

Twitchのアクセストークンは約4時間で失効します。  
**Client Secret** を設定している場合は自動的にリフレッシュされます。  
未設定の場合、失効時に再認証が必要になります。

### アンケート・予測の制限

アンケート（Poll）と予測（Prediction）は **Twitchアフィリエイトまたはパートナー** のみ利用可能です。  
一般ユーザーの場合、これらの機能は自動的にグレーアウトされます。

### コメントの大量送信

視聴者が多い配信では、コメントが大量にAIに送信されます。  
Bot除外機能を活用して、不要なコメントを除外してください。

### 接続モードの切り替え

- 認証モードからIRCモードに切り替えると、認証モードの接続は自動的に切断されます
- IRCモードで接続中はモード切り替えがロックされます（切断してから切り替え）
