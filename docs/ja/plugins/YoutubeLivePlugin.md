# 📺 YouTube Live (YoutubeLivePlugin.py)

YouTube Liveの配信URLを指定するだけで、**リスナーのコメントをリアルタイムに拾ってAIに反応させる**ことができるプラグインです。

配信の「タイトル」「説明文」「サムネイル画像」も自動的に取得してAIに送り込むため、AIが配信内容を理解した上で番組を盛り上げてくれます。

---

## ⚙️ 使い方

### 1. 設定画面を開く
TeloPonのメイン画面右側、「拡張機能」パネルにある **「📺 YouTube Live」** の **「🖥️ 操作パネル」** ボタンをクリックします。

### 2. URLを入力して「接続」する
上部の入力欄にYouTube LiveのURLを貼り付け、**「接続」** ボタンを押します。  
※ 視聴者が見る公開用URLを入力してください。

### 3. 接続確認
接続に成功すると「✅ 接続成功」と表示され、配信のタイトル・説明文・サムネイルがプレビューされます。

### 4. ライブ接続を開始する
TeloPonのメイン画面で **「🔴 ライブ接続開始」** ボタンを押してAIとの通話を開始します。

---

## 🧠 AIに連携される情報

| タイミング | 内容 |
|---|---|
| 接続時 | 配信タイトル・説明文がAIのプロンプトに追加 |
| 通話開始5秒後 | サムネイル画像がAIの視覚に送信 |
| 随時 | 視聴者のリアルタイムコメントを自動送信 |

---

## 🎤 サムネイル音声コマンド

配信プラットフォームに接続すると、サムネイル画像をOBSに表示する音声コマンドが使えます。

| 音声例 | 動作 |
|---|---|
| 「サムネ出して」「サムネイル表示して」 | OBSにサムネイル画像を小さく表示 |
| 「サムネ大きくして」 | サムネイルを拡大表示 |
| 「サムネ小さくして」 | サムネイルを縮小表示 |
| 「サムネ消して」 | サムネイルを非表示 |

---

## ⚠️ ご利用上の注意

- **接続順序**: TeloPonの「ライブ接続開始」前にURLに接続しておくと、タイトルや説明文がAIに確実に認識されます。
- **コメントが多すぎる場合**: AIがすべてのコメントに反応しようとしてパンクする可能性があります。
- **限定公開**: URLを知っていれば接続可能です。非公開の配信には接続できません。
