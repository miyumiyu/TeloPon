# 🚀 Release V2.10b：麻雀牌表示・アバター対応・安定性強化！

> 👉 **[TeloPon v2.10b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.10b/TeloPon-210b.zip)**

今回のアップデートは、**Gemini 3.1 Flash Live のさらなる安定化・最適化**を軸に、麻雀配信サポートや視聴者アバター表示など多彩な機能を追加しました。

---

## ✨ 主な追加・改善

### ⚡ Gemini 3.1 Flash Live 最適化・安定化

Gemini 3.1 Flash Live 向けの接続処理を全面的にリファクタリングしました。

* テロップの応答速度がさらに向上し、よりリアルタイムに近い表示が可能になりました
* セッション再接続の安定性を改善

---

### 🆕 視聴者コメント返信の強化（window-reply）

コメント返信テロップ `window-reply` のデザインと機能を大幅に強化しました。

* **吹き出しスタイル**に変更し、画面**左上**に配置
* **YouTube・Twitch**のコメント投稿者サムネイルを返信テロップに表示
  * YouTubeはコメント取得時に自動でアバターを取得します
  * TwitchはHelix APIを使用します（Client ID・Secret IDを設定した場合のみ）→ [Twitchプラグイン設定](../plugins/TwitchPlugin.md)

---

### 🆕 麻雀配信サポート

麻雀配信向けの新機能を追加しました。

#### 麻雀牌インライン表示タグ

プロンプト内に牌タグを記述すると、テロップ上に牌の画像が自動で表示されます。

| 種類 | タグ |
|---|---|
| 萬子 | `<M1>` 〜 `<M9>` |
| 索子 | `<S1>` 〜 `<S9>` |
| 筒子 | `<P1>` 〜 `<P9>` |
| 字牌 | `<J1>`東 `<J2>`南 `<J3>`西 `<J4>`北 `<J5>`白 `<J6>`発 `<J7>`中 |

#### 麻雀アドバイザープロンプト `r_mahjong.txt`

* OBSキャプチャと組み合わせて、手牌の捨て牌アドバイスをテロップで表示できます
* 画面に見えている牌のみを根拠にアドバイスするよう設計しています
* ⚠️ **AIは画像の読み取りを誤ることがあり、存在しない牌を言及する場合があります。** エンタメとしてお楽しみください

---

### 🔧 安定性・その他の改善

* AI生成直後の「沈黙誤認」によるテロップ暴発を防止
* その他軽微なバグ修正
