# 🚀 Release V2.13b：プラグインテロップ出力フック・TelopViewerプラグイン追加！

> 👉 **[TeloPon v2.13b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.13b/TeloPon-213b.zip)**

今回のアップデートでは、プラグイン機構にテロップ出力フックを追加し、AIの出力をプラグインで受け取れるようになりました。また、従来のテロップモニターを廃止し、同等の機能をプラグインとして再構築しました。

---

## ✨ プラグインにテロップ出力フック機構を追加

### 🆕 on_telop_output フック

AIがテロップを出力するたびに、プラグインがそのデータをリアルタイムに受け取れる `on_telop_output` フックを新設しました。

* プラグインは **topic / main / window / layout / badge** の全データを受け取れます
* `<b1>` `<b2>` タグや麻雀牌タグ（`<P1>` 等）も生データのまま渡されます
* 戻り値でOBSテロップ表示を制御できます

| 戻り値 | 動作 |
|---|---|
| `0` | 即表示（デフォルト） |
| `1` 以上 | N秒後に表示（ディレイ） |
| `-1` | OBS表示を抑制（プラグインにはデータが届く） |

### 🛡️ 安全機構

プラグインの不具合がシステム全体に影響しないよう、以下の保護機構を搭載しています。

* 各プラグインの `on_telop_output` は **タイムアウト1秒** で実行
* **3回タイムアウト** したプラグインは自動的にフック呼び出し対象から除外

👉 **[プラグイン開発ガイド（on_telop_output の詳細）](https://github.com/miyumiyu/TeloPon/blob/main/docs/ja/03_plugin_dev.md)**

## ✨ テロップモニター廃止 → TelopViewer プラグインで代替

### 🆕 TelopViewer プラグイン

従来の「テロップモニター」機能を廃止し、`on_telop_output` フックを活用した **TelopViewer プラグイン** として再構築しました。

* AIが出力したテロップ内容を **操作パネルでリアルタイム表示**
* **B1/B2タグの色分け表示**（B1=赤系、B2=シアン系）
* **表示ディレイをプルダウンで設定** 可能（0/1/2/3/5/10/-1）
* プラグイン一覧の「操作パネル」ボタンから開くだけで使えます
