# 🚀 Release V2.11b：テロップ縦積みスタック・ズーム・アンカー移動対応！

> 👉 **[TeloPon v2.11b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.11b/TeloPon-211b.zip)**

今回のアップデートは、**OBS上のテロップ表示をより柔軟にコントロールできる**よう、縦積みスタック・ズーム・アンカー移動などの表示UI機能を大幅に強化しました。  
(🙏 Special Thanks **[@mo9mo9ke6ri](https://x.com/mo9mo9ke6ri)**)


---

## ✨ 主な追加・改善

### 🆕 テロップ縦積みスタック

複数のテロップが重ならないよう、自動的に縦に積み上げて表示するようになりました。

* テロップは **`window-reply` グループ**（左側）と**通常グループ**（右側）の2つに分かれて独立管理されます
* 各グループ最大 **4枚** まで同時表示
* 新しいテロップが上（または下）に出現し、古いものがスライドして移動します
* 古いテロップは徐々に透明になります（100% → 50% → 30% → 10%）

---

### 🆕 アンカー移動（ドラッグで出現位置を変更）

テロップをドラッグすると、以降のテロップがその位置を基準に出現するようになりました。

* **アンカー位置はブラウザの `localStorage` に保存**され、OBS再起動後も維持されます
* アンカーのY座標が画面下3/4より下にある場合、スタック方向が**自動的に上向き**に切り替わります

---

### 🆕 ホイールズームでテロップサイズを変更

テロップにマウスカーソルを合わせてホイール操作すると、テロップのサイズを変更できます。

* ズーム倍率は **`window-reply` グループ** と **通常グループ** それぞれ独立して保存されます
* 次に同グループのテロップが出現するとき、保存済みのサイズで表示されます
* **ダブルクリックでサイズをリセット**できます（位置もリセット）
* OBS での使用には、ブラウザソースの「**インタラクティブ**」設定を有効にしてください
