# 🚀 Release V2.15b：AIアイコン・UI再構成・安定性向上！

> 👉 **[TeloPon v2.15b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.15b/TeloPon-215b.zip)**

今回のアップデートでは、テロップにAIアイコンを表示する機能の追加、UIの再構成、および通信安定性の向上を行いました。

---

## ✨ AIアイコン機能

### 🆕 テロップにAIアイコンを表示

AIの性格に合わせたアイコン画像をテロップの左側に表示できるようになりました。

* UIの「AIアイコン選択」ボタンから画像を設定
* **枠の高さに合わせて自動リサイズ**（正方形・中心トリミング）
* 画像がない場合は従来通りの表示
* 対象：variety / news / digital / cute / simple
* reply / explain / nameplate には表示されません

---

## ✨ UI再構成

設定画面のレイアウトを見直し、より分かりやすく整理しました。

* **⚙️ AI設定**: API Key・認証状態・モデル・思考レベル・AIの性格・AIアイコン
* **📡 配信設定**: 配信者名・マイク選択・メーター・閾値・テーマ

---

## 🔧 安定性の向上

### AIの無応答に対する自動復帰機構を追加

AIからの応答が長時間途絶えた場合に、自動的にセッションを再接続する機構を追加しました。
配信中にAIが沈黙し続ける問題の発生頻度が大幅に低減されます。

---

## 🔧 その他の修正

* v2.14bで一部のプラグインが動作しなくなっていた問題を修正（ctypes.wintypes をビルドに追加）
* テロップ表示ディレイが起動時に前回の値を引き継いでしまう問題を修正（起動時は常に0にリセット）
