# 🎙️ TeloPon V2.20b — 「声だけで、配信のすべてを操る。」

> 👉 **[TeloPon v2.20b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.20b/TeloPon-220b.zip)**

---

## 🌟 ハイライト

配信プラットフォームのAPIに完全対応。YouTube Live・Twitch・ニコニコ生放送の
アンケート作成・コメント書き込み・タイトル変更・視聴者数取得など、
**プラットフォーム特有の配信機能を、AIに話しかけるだけで操作**できるようになりました。

プラグインマネージャーの搭載により、必要な機能をワンクリックで追加。
拡張プラグインは専用リポジトリに分離し、サムネイル表示・OBS録画・ツイート投稿まで、
声ひとつで配信のすべてをコントロールする体験をお届けします。

---

## 🆕 新機能

### 📺 配信プラットフォームAPI完全対応
- **YouTube Live**: コメント読み書き・アンケート作成/集計・配信タイトル変更・視聴者数取得・スーパーチャット対応（OAuth連携）
- **Twitch**: コメント読み書き・アンケート・予測（ベッティング）・クリップ作成・タイトル変更・視聴者数取得（OAuth2 Device Code Flow）
- **ニコニコ生放送**: コメント取得・ギフト/ニコニ広告通知・アンケート作成/集計・運営コメント投稿・視聴者数取得（同梱プラグインに昇格）

### 🔧 プラグインマネージャー
- プラグインの有効/無効をUIから切替
- 拡張プラグインをワンクリックでダウンロード＆即利用
- 自作プラグインの手動インストールにも対応

### 🎤 サムネイル音声操作
- 「サムネ出して」「大きくして」「消して」でOBSにサムネイル画像を表示/操作
- 配信プラットフォーム接続時に自動で有効化

### 🎮 OBS連携の強化
- OBS録画の音声操作（「録画開始して」「録画止めて」）
- キャプチャ対象をシーン一覧からプルダウン選択

### 🗣️ システム音声コマンド
- 「説明消して」「ネームプレート閉じて」でAIが画面上のウィンドウを閉じます

---

## 📦 拡張プラグイン

拡張プラグインは専用リポジトリで管理されるようになりました。  
プラグインマネージャーからワンクリックでダウンロードできます。

🔗 **[TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)**

| プラグイン | 概要 |
|---|---|
| 📺 **YouTube Live+** | YouTube OAuth連携（コメント読み書き・アンケート・タイトル変更・スーパーチャット） |
| 🐦 **X (Twitter)** | ハッシュタグ取得・サムネ/画面付きツイート投稿 |
| 🔊 **テロップ読み上げ (Windows)** | Windows標準音声で自動読み上げ |
| 🔊 **テロップ読み上げ (VOICEVOX)** | VOICEVOX音声合成 |
| 📊 **PowerPoint操作** | スライドショー音声操作・ノート自動注入 |
| 💬 **Discord** | Discordチャンネルコメント取得 |
| 💬 **Slack** | Slackチャンネルコメント取得 |
| 🎮 **VCI テロップ送信** | VirtualCast VRテロップ |

## 🔄 改善・変更

- プラグイン名を統一命名規則に変更（📺配信/🐦SNS/💬チャット/🔊読み上げ/🎮画面/📊プレゼン）
- 配信情報共有機構を追加（配信URL・サムネイルをプラグイン間で共有）
- VCI OSCプラグインのUTF-8文字化け修正
- カンペ・画像注入のUIを簡素化
- 麻雀牌画像がちらつく問題を修正
- AIが各国の標準時間を正しく認識するように改善（日本:JST / 韓国:KST / ロシア:MSK）
- ドキュメント全面改修（4言語）
