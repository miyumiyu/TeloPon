# TeloPon V2.16b リリースノート

## 🎮 Twitch連携プラグインを大幅強化

OAuth2認証に対応し、コメント読み取りだけだったTwitch連携が全面進化しました。

- **コメント書き込み** - 「○○ってチャットに書いて」でAIがTwitchチャットに投稿
- **アンケート** - 「好きな色をアンケートして」で投票を作成、「集計して」で結果発表
- **予測（ベッティング）** - 「このボス倒せるか賭けさせて」で視聴者がポイントを賭ける（アフィリエイト/パートナー）
- **クリップ作成** - 「今の面白かった！クリップして」で直前30秒を保存
- **タイトル変更** - 「タイトルを雑談配信に変えて」で即変更
- **視聴者数取得** - 「今何人見てる？」でAIが教えてくれる
- **Bot除外** - nightbot等をダブルクリックで除外リストに追加
- **認証モード/IRCモード** - 全機能 or コメント読取のみ（認証不要）を切替可能

## ▶️ 拡張プラグインに YouTube OAuth連携 を追加

YouTube Data API + OAuth2 による上位互換 YouTube 連携プラグインをリリース。

- **コメント書き込み** - 「○○ってコメントして」でAIがYouTubeチャットに投稿
- **アンケート** - 「アンケートして」でYouTube公式アンケートを作成、「集計して」で結果をAIが発表
- **タイトル変更** - 「タイトルを○○にして」で配信タイトルを変更
- **視聴者数取得** - 「今何人？」で同時視聴者数を取得してAIが教える
- **配信選択** - 予約済み配信の一覧から選択、またはURLで直接指定
- **権限自動判定** - オーナー/管理者かどうかを自動判定し使える機能を表示

GCP設定ガイド（画像付き）も用意しました。

👉 [TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)

## 🛡️ Twitch / YouTube OAuth連携にコマンド使用権限を追加

Twitch連携・YouTube OAuth連携の各機能に「視聴者可」チェックボックスを追加。  
視聴者のコメントからコマンドを実行させるかどうかを機能ごとに個別に制御できます。  
例えば、タイトル変更やアンケートは配信者の音声のみで実行し、視聴者コメントからは実行しない、といった設定が可能です。

## 🎤 説明枠・ネームプレートを音声で閉じる

「説明消して」「ネームプレート閉じて」でAIが画面上のウィンドウを閉じます。
