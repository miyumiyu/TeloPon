# 🎙️ TeloPon V2.23b — 「AIが時計を持った日。」

> 👉 **[TeloPon v2.23b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.23b/TeloPon-223b.zip)**

---

## 🆕 新機能

### 🕐 正確な時刻応答（Function Calling）
「今何時？」「今日何日？」と聞くと、AIがシステム時刻を取得して正確な日本時間（JST）で回答します。  
Gemini Live API の Function Calling を活用し、推測や記憶ではなくリアルタイムの時刻データを参照します。

### 📺 サムネイル・説明文のAI注入ON/OFF
YouTube Live・YouTube Live+・Twitch・ニコニコ生放送の各プラグインに、配信開始時にAIへ送るサムネイル画像と説明文のON/OFFチェックボックスを追加しました。  
不要な場合にOFFにすることで、プロンプトのトークンを節約できます。

### 💬 テロップをTwitchチャットに投稿
Twitchプラグインの機能設定に「テロップをチャットにも投稿」チェックボックスを追加。  
ONにすると、AIが出力するテロップの内容がTwitchチャットにも自動で投稿されます。
