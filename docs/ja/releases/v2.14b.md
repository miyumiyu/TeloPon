# 🚀 Release V2.14b：プラグイン疎結合化・テロップ視認性改善・VOICEVOX読み上げ追加！

> 👉 **[TeloPon v2.14b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.14b/TeloPon-214b.zip)**

今回のアップデートでは、プラグインアーキテクチャの疎結合化、テロップの視認性改善、そして拡張プラグインにVOICEVOX読み上げを追加しました。

---

## ✨ プラグインアーキテクチャの疎結合化

### 🆕 fire-and-forget 方式

テロップ出力フック（`on_telop_output`）を **fire-and-forget（投げっぱなし）** 方式に変更しました。

* Core はプラグインの処理完了を **一切待ちません**
* プラグイン内でエラーや遅延が発生しても **システム全体に影響しません**
* ディレイ制御は `set_telop_delay()` / `get_telop_delay()` で分離

これにより、サードパーティのプラグインが安全に動作する基盤が整いました。

👉 **[プラグイン開発ガイド（詳細）](https://github.com/miyumiyu/TeloPon/blob/main/docs/ja/03_plugin_dev.md)**

---

## ✨ テロップ視認性の改善

普通枠テロップのスタック表示時の透明度を変更し、古いテロップの視認性を向上しました。

| 順番（新着→古い） | 変更前 | 変更後 |
|---|---|---|
| 1番目（最新） | 100% | 100% |
| 2番目 | 50% | **75%** |
| 3番目 | 30% | **50%** |
| 4番目以降 | 10% | **25%** |

---

## ✨ 拡張プラグインに VOICEVOX 読み上げを追加

### 🆕 VOICEVOX 読み上げプラグイン

AIが出力したテロップのテキストを **VOICEVOX** の音声合成エンジンでリアルタイムに読み上げる拡張プラグインを追加しました。

* **キャラクター＆スタイル選択**: VOICEVOXの全キャラクター＋性格（ノーマル/あまあま/ツンツン等）をプルダウンで選択
* **アイコン表示**: 選択したキャラクター・スタイルのアイコンが設定画面に表示
* **5つの音声パラメータ**: 話速・音量・音高・抑揚・発話前無音をリアルタイム調整
* **TOPIC読み上げ**: ONにするとテロップの上段テキストも一緒に読み上げ
* **テロップ表示制御**: 読み上げのみ（テロップ非表示）やディレイ表示が可能
* **自動接続確認**: VOICEVOXへの接続を自動確認。接続できない場合はUIがグレーアウト

👉 **[VOICEVOX読み上げプラグイン 詳細ガイド](https://github.com/miyumiyu/TeloPon/blob/main/docs/ja/plugins/voicevox_plugin.md)**

> 💡 別途 [VOICEVOX](https://voicevox.hiroshiba.jp/) のインストールが必要です。
