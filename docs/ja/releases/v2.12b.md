# 🚀 Release V2.12b：replyカードデザイン刷新・explain位置記憶対応！

> 👉 **[TeloPon v2.12b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

今回のアップデートは、**`window-reply` のビジュアルをSNSカード風にリデザイン**し、アバター表示・スライドアニメーション・グロー演出を追加しました。また、説明枠（`window-explain`）の位置・スケールがOBS再起動後も保持されるようになりました。

---

## ✨ 主な追加・改善

### 🆕 window-reply SNSカード風デザイン

`window-reply` テロップがSNSのリプライカードのような見た目にリデザインされました。

* **投稿者アバターアイコン**を円形で表示（YouTube等のアイコン画像に対応）
* アバター右下に **↩ 返信バッジ** を表示
* テロップ出現時に**スライドインアニメーション**（左右どちらから来るかはアンカー位置で自動判定）
* ダーク背景＋**グロー（光彩）エフェクト**（全4テーマ対応：default / comic / honwaka / tsundere）

---

### 🆕 explain 位置・スケールの永続化

`window-explain`（説明枠）の位置とズーム倍率が `localStorage` に保存されるようになりました。

* ドラッグで移動した位置が **OBS再起動後も維持**されます
* ホイールでズームしたスケールも**保持**されます
* **ダブルクリックで位置・スケールをリセット**できます

---

### 🔧 その他の修正

* `window-reply` スライドアニメーション後に保存済みスケールが適用されなかった問題を修正
* ダブルクリックでリセット時、テロップが一瞬左上に飛ぶ問題を修正
* OBSブラウザソースがCSSをキャッシュし続ける問題を修正（サーバーのno-cacheヘッダー＋URLタイムスタンプ）
