# 🚀 Release V2.12b：テロップUI/UX全面改善・思考レベル設定対応！

> 👉 **[TeloPon v2.12b をダウンロード](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

今回のアップデートでは、テロップのビジュアルと操作性を全面的に改善しました。また、Gemini 3.1に対応した思考レベルの設定が可能になりました。

---

## ✨ テロップUI/UXの全面改善

### 🆕 window-reply SNSカード風デザイン

`window-reply` テロップがSNSのリプライカードのような見た目にリデザインされました。

* **投稿者アバターアイコン**を円形で表示（YouTube等のアイコン画像に対応）
* アバター右下に **↩ 返信バッジ** を表示
* テロップ出現時に**スライドインアニメーション**（左右どちらから来るかはアンカー位置で自動判定）
* ダーク背景＋**グロー（光彩）エフェクト**（全4テーマ対応：default / comic / honwaka / tsundere）

---

### 🆕 アニメーションの滑らか化

ドラッグによる移動、テロップの出現・消滅アニメーションを全面的に改善しました。

* テロップ出現：**scale(0)から滑らかに拡大**して出現
* テロップ消滅：**scale(0)まで滑らかに縮小**して消滅（従来の半分サイズで突然消える挙動を修正）
* ドラッグ移動：**遅延やカクつきのない即座の追従**
* キュー切替：次のアイテムがある場合、**フェードアウトとフェードインがオーバーラップ**するシームレスな切替

---

### 🆕 explain「次へ」バッジ / nameplate「×」閉じるボタン

テロップの操作性を向上するボタンを追加しました。

* **explain 枠**：キューにアイテムがある場合、右下にキュー数＋「›」バッジを表示。クリックで即座に次のアイテムへ
* **nameplate 枠**：右上に「✕」閉じるボタンを常時表示。クリックで即座に消去

---

### 🆕 explain ドラッグ改善

`window-explain`（説明枠）のドラッグ操作を大幅に改善しました。

* ドラッグで移動した位置が **OBS再起動後も維持**されます
* ホイールでズームしたスケールも**保持**されます
* **ダブルクリックで位置・スケールをリセット**できます
* ビューポート端に押すと枠が**水平方向に圧縮**、戻すと拡大（左右40px固定マージン）
* **ドラッグ中はタイマーを一時停止**（ドラッグ中にテロップが消えません）

👉 **[OBSでのテロップ操作の詳しい説明はこちら](https://github.com/miyumiyu/TeloPon/blob/main/docs/ja/05_obs_control.md)**

## ✨ 思考レベルの設定

### 🆕 Gemini 3.1 思考レベル対応

Gemini 3.1の新しい思考機能に対応しました。UIの「💭 思考レベル」プルダウンから設定できます。

| レベル | 説明 |
|---|---|
| **minimal** | 最低レイテンシ優先（デフォルト） |
| **low** | シンプルなタスク向け |
| **medium** | 中程度の複雑さ向け |
| **high** | 複雑な推論・多段プランニング向け |

* 従来の `thinking_budget`（トークン数指定）は廃止されました
* 設定は `settings.json` に保存され、次回起動時も復元されます
