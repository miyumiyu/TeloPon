# 🎛️ Release V1.22b：自動割込モード新搭載・UI改善！

今回のアップデートの目玉は**「自動割込」**の新機能追加です。
あわせて発話モードの選択UIをリニューアルし、より使いやすくなりました。

---

## ✨ 主な追加・改善

### 🆕 新機能：自動割込モード

前回テロップが出てから一定時間が経過したとき、AIがここまでの話題を自動的に要約・整理するテロップを出してくれる機能です。

* UIの「発話モード」コンボボックスで **「自動割込」** を選択して有効化
* 「介入時間（秒）」スライダーで発動間隔を設定
* コンボボックス横に**発動までのカウントダウン**がリアルタイム表示されます

**こんな場面に最適：**
ニュース・情報系配信や長い雑談セッションで、定期的にAIがトピックをまとめてくれるため、視聴者も話の流れを追いやすくなります。介入時間は 60〜120秒 がおすすめです。

---

### 🔄 UI改善：発話モードをコンボボックスで選択

従来のチェックボックスによる「自動介入ON/OFF」から、**コンボボックスで発話モードを選択する形式**に変更しました。

| モード | 動作 |
|---|---|
| **None** | 自動介入なし |
| **自動発話** | 話者が沈黙してからN秒後にAIが自発的に話題を振る（従来のAuto Talk） |
| **自動割込** | 前回テロップが出てからN秒後にAIが話題を要約・整理する（新機能） |

* コンボボックス横のカウントダウン表示で、**発動までの残り秒数**が一目でわかります
* AI思考中もカウントダウンが止まらず、正確な発動タイミングを把握できます
* ライブ接続中でもモードを切り替えられます

---

### 🐛 バグ修正・安定性向上

* 短時間のノイズ（咳払い・物音）が発話として誤検知され、空ターンが頻発する問題を修正
  （0.3秒以上継続した音声のみ発話として扱うよう改善）
* 喋っているにも関わらず、AIが無言と認識してしまう問題を修正
