# 🌐 Release V1.20b：4言語対応 & Twitch連携プラグイン同梱！

今回のアップデートは「**グローバル対応**」と「**配信プラットフォームの拡充**」をメインとしたバージョンです。
UIとドキュメントを一気に4言語対応し、海外配信者にも使いやすいアシスタントツールへと進化しました。
さらに、Twitchのチャット読み取りと配信情報連携が標準プラグインとして同梱され、YouTubeだけでなくTwitchでの配信もすぐに始められます！

---

## ✨ 主な新機能

### 🌐 UIの4言語対応（日本語 / English / 한국어 / Русский）

TeloPon本体のUI（ボタン・ステータス・メニュー等）が、OSの言語設定に応じて自動的に切り替わるようになりました。

* **日本語**・**英語**・**韓国語**・**ロシア語** に対応。
* 起動時にOSの言語を検出し、非対応言語の場合は英語にフォールバックします。
* プラグインのUI表示も同様に多言語対応済みです。

---

### 🎮 Twitch連携プラグインを標準同梱（TwitchPlugin.py）

Twitchでの配信をサポートする新プラグインが、インストール不要の**標準同梱プラグイン**として追加されました。

* **チャット読み取り**：チャンネル名を入力するだけで、視聴者のコメントをリアルタイムに取得してAIへ渡します。**Client IDなしで動作**します。
* **ストリーム情報の注入**：Twitch Developer ConsoleでClient ID / Client Secretを設定すると、配信の「タイトル」「カテゴリ」「サムネイル」もAIに自動送信されます。
* **URL / チャンネル名どちらでも入力OK**：`https://twitch.tv/mychannel` 形式でも `mychannel` 単体でも認識します。

👉 [Twitch連携ツール 詳細な使い方](../plugins/TwitchPlugin.md)

---

## 💡 配信者様へのワンポイントアドバイス

**TwitchとYouTubeを同時に配信している方へ**
TwitchプラグインとYouTubeプラグインは同時に有効にできます。両方のチャットを同時にAIへ渡すことで、マルチプラットフォーム配信でも視聴者全員のコメントにAIが反応する、"全員参加型"の配信体験が実現できます！

**Twitchプラグインはすぐ試せます！**
Client IDの取得は省略できます。チャンネル名を入れて「接続」→「🔴ライブ接続開始」だけで、Twitchのコメント連携をすぐに試せます。
