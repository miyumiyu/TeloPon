# 🎮 OBSでテロップを操作する方法

TeloPonのテロップはOBS上で自由に**ドラッグ移動**や**拡大・縮小**ができます。
このページでは、OBSブラウザソース上でテロップを操作する方法を説明します。

---

## 1. OBSの「操作」モードに入る

OBSのプレビュー画面の下にある **「操作（対話）」ボタン** をクリックします。

![OBS操作ボタン](../../images/obsctl.png)

> ボタンをクリックすると、プレビュー画面が赤い枠で囲まれ、ブラウザソースへの操作が有効になります。

---

## 2. テロップの操作方法

「操作」モードに入ると、テロップを直接マウスで操作できます。

![OBS操作中の画面](../../images/obsctl2.png)

### ドラッグで移動

| 対象 | 操作 |
|---|---|
| **通常テロップ**（variety / news / reply 等） | テロップをドラッグして好きな位置へ移動。位置はOBS再起動後も記憶されます |
| **説明枠**（explain） | ドラッグで移動。位置はOBS再起動後も記憶されます |
| **ネームプレート**（nameplate） | 画面下部に表示。閉じるには右上の「✕」をクリック |

### ホイールで拡大・縮小

テロップの上で**マウスホイール**を回すと、拡大・縮小できます。

* **ホイール上** → 拡大
* **ホイール下** → 縮小
* スケールはグループ単位で記憶されます

### ダブルクリックでリセット

テロップを**ダブルクリック**すると、位置とスケールが初期状態にリセットされます。

### 長押しで消去

テロップを**1秒以上長押し**すると、そのテロップを手動で消去できます。

---

## 3. 説明枠（explain）の特殊操作

説明枠にはいくつかの追加機能があります。

| 操作 | 動作 |
|---|---|
| **ドラッグ** | 自由に移動（位置はOBS再起動後も記憶） |
| **ホイール** | 拡大・縮小（スケールも記憶） |
| **ダブルクリック** | 位置・スケールをリセット |
| **長押し（1秒）** | 現在の説明枠を消去して次へ進む |
| **端へドラッグ** | 画面端に押すと枠が水平方向に圧縮。戻すと復元 |
| **「次へ」バッジ** | キューにアイテムがある場合、右下にバッジが表示。クリックで次のアイテムへ |
