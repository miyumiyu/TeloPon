# 🔊 VOICEVOX 읽기 (voicevox_plugin.py)

이 플러그인은 AI가 출력한 텔롭의 텍스트를 **VOICEVOX** 음성 합성 엔진으로 읽어줍니다.
다양한 캐릭터 보이스와 세밀한 음성 조정으로 방송을 더욱 풍성하게 연출할 수 있습니다.

---

## 🛠️ 준비: VOICEVOX 설치 및 실행

TeloPon과 별도로 VOICEVOX를 설치하고 실행해야 합니다.

1. **[VOICEVOX 공식 사이트](https://voicevox.hiroshiba.jp/)** 에서 VOICEVOX를 다운로드·설치합니다.
2. **VOICEVOX를 실행** 합니다. 실행하면 내부에 HTTP 서버（기본: `http://localhost:50021`）가 시작됩니다.
3. VOICEVOX는 **TeloPon과 동시에 실행** 한 상태로 두세요.

> 💡 VOICEVOX 창은 최소화해도 됩니다. 백그라운드에서 동작합니다.

---

## ⚙️ TeloPon 설정 및 사용법

### 1. 조작 패널 열기

TeloPon 메인 화면 오른쪽 「확장 기능（플러그인）」 패널에서 **「VOICEVOX 읽기」** 의 **「조작 패널」** 버튼을 클릭합니다.

![VOICEVOX 읽기 설정 화면](../../../images/voicevox_plugin.png)

---

### 2. 연결 테스트

1. **URL** 에 VOICEVOX 주소를 입력합니다（기본: `http://localhost:50021`）
2. **「연결 테스트」** 버튼을 누릅니다
3. 「✅ 연결 성공」이 표시되면 준비 완료입니다

> ⚠️ 연결에 실패하면 VOICEVOX가 실행 중인지 확인하세요. 연결할 수 없는 상태에서는 음성 설정이 비활성화됩니다.

### 3. 읽기 ON

**「읽기 ON」** 체크박스를 체크하면 텔롭이 나올 때마다 자동으로 읽어줍니다.

### 4. 캐릭터와 스타일 선택

연결 테스트 성공 후 VOICEVOX에 등록된 캐릭터 목록이 드롭다운에 표시됩니다.

* **캐릭터**: 읽기에 사용할 보이스 캐릭터를 선택합니다（예: 시코쿠 메탄, 즌다몬 등）
* **스타일**: 같은 캐릭터에도 여러 스타일이 있습니다（예: 노멀, 상냥, 츤츤 등）

캐릭터/스타일을 선택하면 왼쪽에 아이콘 이미지가 표시됩니다.

### 5. 음성 파라미터 조정

| 파라미터 | 범위 | 기본값 | 설명 |
|---|---|---|---|
| **속도** | 0.5 〜 2.0 | 1.0 | 읽기 속도. 클수록 빠름 |
| **음량** | 0.1 〜 2.0 | 1.0 | 읽기 음량 |
| **음높이** | -0.15 〜 0.15 | 0.0 | 목소리 높이. +로 높게, -로 낮게 |
| **억양** | 0.0 〜 2.0 | 1.0 | 감정 강약. 0이면 단조로움, 클수록 감정 풍부 |
| **발화 전 무음** | 0.0 〜 1.5초 | 0.1 | 읽기 시작 전 간격（초） |

> 💡 파라미터는 다음 텔롭 읽기부터 즉시 반영됩니다.

### 6. 텔롭 설정

* **📌 TOPIC도 읽기**: ON이면 「TOPIC。MAIN」형태로 읽습니다. OFF이면 MAIN만.
* **📺 텔롭 표시**: ON이면 OBS에 텔롭 표시（딜레이 설정 가능）. OFF이면 텔롭 표시 억제.
* **텔롭 딜레이（초）**: OBS 텔롭 표시를 지연시킵니다（0 = 즉시 표시）.

### 7. 닫기

**「닫기」** 버튼 또는 창의 **×** 버튼으로 설정 패널을 닫습니다. 설정은 자동으로 `plugins/voicevox.json`에 저장되어 다음 실행 시 복원됩니다.

## ❓ 문제 해결

### Q. 연결 테스트에서 「❌ 연결 실패」가 나옵니다
- VOICEVOX가 실행 중인지 확인하세요
- URL이 `http://localhost:50021`인지 확인하세요
- 보안 소프트웨어가 포트 50021을 차단하고 있지 않은지 확인하세요

### Q. 읽기 중에 갑자기 멈췄습니다
- VOICEVOX가 종료되었을 수 있습니다. 읽기 중 연결이 끊기면 자동으로 OFF됩니다
- VOICEVOX를 재시작하고 조작 패널의 「연결 테스트」로 다시 연결하세요

### Q. 소리가 들리지 않습니다
- PC 볼륨이 음소거되어 있지 않은지 확인하세요
- 「읽기 ON」체크박스가 체크되어 있는지 확인하세요
- 음량 파라미터가 0.1 이상인지 확인하세요
