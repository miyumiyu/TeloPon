# 🎮 OBS 화면 AI 실황 (obs_capture.py)

이 플러그인은 **특정 OBS Studio 화면(게임 캡처, 카메라 피드 등)을 주기적으로 캡처하여 자동 또는 수동으로 AI에게 보여주는** 매우 강력한 도구입니다.

스트리머의 외모나 현재 플레이 중인 게임에 대한 코멘트 등 — **시각 정보를 기반으로 한 실시간 AI 실황 및 반응**을 가능하게 합니다.

---

## 🛠️ 설정: OBS 설정 (OBS WebSocket)

TeloPon이 OBS 화면을 캡처할 수 있도록 먼저 OBS에서 "WebSocket" 기능을 활성화해야 합니다.

1. **OBS Studio**를 실행합니다.
2. 상단 메뉴에서 **"도구"**, **"WebSocket 서버 설정"**을 클릭합니다.
![OBS 캡처 설정](../../../images/obs_capture02.png)

3. **"WebSocket 서버 활성화"**를 체크합니다.
4. **서버 포트**는 기본값 `4455`로 유지합니다.
5. **"인증 활성화"**를 체크하고 원하는 **비밀번호**를 설정합니다 (TeloPon에 입력해야 하므로 기억해 두세요).
6. "적용" → "확인"을 눌러 닫습니다.

## ⚙️ TeloPon 설정 및 사용법

### 1. 제어 패널 열기
TeloPon 메인 화면 우측의 "확장(플러그인)" 패널에서 **"OBS 화면 AI 실황"**의 **"🖥️ 제어 패널"** 버튼을 클릭합니다.

![OBS 화면 AI 실황 제어 패널](../../../images/obs_capture01.png)

### 2. OBS WebSocket 연결 설정
OBS에서 설정한 정보를 입력합니다.
* **호스트 / 포트**: 일반적으로 `127.0.0.1`과 `4455`로 유지합니다.
  (OBS와 TeloPon이 같은 PC에서 실행 중인 경우)
* **비밀번호**: OBS 측에서 설정한 비밀번호를 입력합니다.

### 3. 캡처 대상 지정 및 테스트
* **대상 소스 이름**: AI에게 보여주려는 OBS 소스의 정확한 **"소스 이름"**을 입력합니다.
  *(예: OBS 소스 목록에 표시된 정확한 이름(예: "게임 캡처" 또는 "화면 캡처")을 입력)*
* **"화면 캡처" 버튼 누르기**: 설정이 올바르면 지정한 소스의 현재 화면이 미리보기 영역에 표시됩니다. 연결 테스트 완료!

### 4. AI 전송 설정
화면을 언제 AI에게 전송할지와 이미지와 함께 보낼 "큐 카드" 텍스트를 설정합니다.

* **자동 전송 간격(초)**
  방송 중 화면을 AI에게 전송하는 빈도. 너무 짧게 설정하면 AI가 계속 말하게 됩니다 — `30`~`60`초 정도가 일반적으로 권장됩니다.
* **이미지와 함께 전송할 지시 텍스트**
  AI가 이미지를 받았을 때 어떻게 반응할지 알려줍니다.
  *(기본값: "[방송 화면 업데이트] 현재 화면입니다! 코멘트나 반응을 해주세요!")*
* **방송 중 자동으로 간격을 두고 AI에게 캡처 및 전송**
  좌상단의 이 체크박스를 체크하면 라이브 연결 중 지정한 간격으로 화면이 AI에게 자동으로 전송됩니다.

완료 후 **"저장"**을 눌러 패널을 닫습니다.

---

## ⚡ 방송 중 사용법 (수동 전송)

자동 전송 외에도, 가챠 결과 화면이나 보스를 쓰러뜨리는 순간처럼 AI가 집중하길 원하는 결정적인 순간에 **수동으로 화면을 AI에게 전송**할 수도 있습니다.

1. TeloPon에서 **"🔴 라이브 연결 시작"**을 눌러 AI 세션을 시작합니다.
2. "OBS 화면 AI 실황"의 **"🖥️ 제어 패널"**을 열어두세요.
3. AI에게 보여주고 싶은 화면이 나타나면 패널 하단의 주황색 **"⚡ 지금 AI에게 화면 전송(강제 전송)"** 버튼을 클릭합니다!
4. AI가 즉시 화면을 읽고 반응할 수 있습니다.

---

## ⚠️ 주의 사항

* **화면 해상도**
  AI에게 전송되는 이미지는 더 빠른 전송을 위해 자동으로 축소됩니다. 결과적으로 화면 가장자리의 매우 작은 텍스트(RPG 상태 수치 등)는 AI가 읽지 못할 수 있습니다.
* **소스를 찾을 수 없음 오류**
  "소스 이름"은 OBS 소스 목록에 표시된 이름과 **정확히 일치해야 합니다** (대소문자 및 공백 포함). 오류가 발생하면 OBS에서 소스 이름을 다시 확인하세요.
