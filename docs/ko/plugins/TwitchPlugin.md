# 🎮 Twitch 라이브 연동 (TwitchPlugin.py)

이 플러그인은 Twitch 채널명 또는 URL을 지정하기만 하면 **시청자의 채팅 댓글을 실시간으로 받아 AI가 반응하게** 할 수 있는 확장 기능입니다.

또한 **Twitch Client ID와 Client Secret** (선택)을 설정하면 방송의 "제목", "카테고리", "썸네일 이미지"도 자동으로 가져와 AI에 전송합니다. **Client ID 없이도 채팅 읽기는 동작합니다.**

---

## ⚙️ 사용 방법

### 1. 설정 화면 열기

TeloPon 메인 화면 오른쪽 "확장 기능 (플러그인)" 패널에서 **"Twitch 라이브 연동"** 의 **"⚙️ 설정"** 버튼을 클릭합니다.

### 2. 채널명 입력

상단 입력란에 Twitch 채널명 (예: `mychannel`) 또는 URL (예: `https://twitch.tv/mychannel`) 을 입력합니다.

### 3. Client ID / Client Secret 설정 (선택)

방송 제목, 카테고리, 썸네일도 AI에 공유하고 싶다면 **"🔑 Client ID"** 와 **"🔒 Client Secret"** 을 입력합니다.

> ⚠️ **미입력 시에도 채팅 읽기는 동작합니다.** 채팅 연동만 사용할 경우 이 단계를 건너뛸 수 있습니다.

**Client ID / Client Secret 취득 방법:**

1. [https://dev.twitch.tv/console](https://dev.twitch.tv/console) 에 Twitch 계정으로 로그인합니다.
2. "애플리케이션 등록"에서 새 앱을 만듭니다 (이름·카테고리는 임의).
3. 표시되는 "Client ID"를 복사하고 "새 시크릿"을 생성하여 입력란에 붙여넣습니다.

### 4. "연결" 버튼 누르기

입력이 완료되면 **"연결"** 버튼을 누릅니다.

* **Client ID 없음**: "✅ 채팅 연결됨"이 표시되며 채팅 읽기가 활성화됩니다.
* **Client ID 있음**: 스트림 정보와 썸네일이 가져와져 표시되고 "✅ 연결됨"이 표시됩니다.

플러그인 배지가 녹색 `ON`으로 바뀐 것을 확인한 후 "닫기"로 화면을 닫습니다.

### 5. 라이브 연결 시작

TeloPon 메인 화면으로 돌아가 **"🔴 라이브 연결 시작"** 버튼을 눌러 AI 세션을 시작합니다.

연동 완료! AI 세션이 시작되면 Twitch 정보가 AI에 공유되고 시청자 댓글에 AI가 자동으로 반응합니다.

---

## 🧠 AI에 전달되는 정보

| 정보 | 조건 | 타이밍 |
|---|---|---|
| 방송 제목 · 카테고리 | Client ID 설정 시 | 라이브 연결 시작 시 |
| 썸네일 이미지 | Client ID 설정 시 | 라이브 연결 시작 약 5초 후 |
| 시청자 채팅 댓글 | 항상 (익명 IRC 경유) | 실시간 (5초마다 일괄 전송) |

### 댓글 주입 동작

* 채팅 댓글은 **5초마다** 또는 **10개가 쌓인 시점**에 일괄로 AI에 전송됩니다.
* 각 댓글은 `[COMMENT] 사용자명: 댓글 내용` 형식으로 AI에 전달됩니다.

---

## ⚠️ 주의 사항

* **연결 순서**
  방송 제목·카테고리를 AI가 확실히 인식하게 하려면 **TeloPon의 "🔴 라이브 연결 시작" 버튼을 누르기 전에 이 도구에서 "연결"해 두는 것**을 강력히 권장합니다. (세션 도중 연결할 경우 제목·카테고리는 인식되지 않습니다.)

* **오프라인 채널**
  Client ID 모드에서 대상 채널이 오프라인 (방송 중이 아닌) 상태일 때는 스트림 정보를 가져올 수 없습니다. 채팅 연결은 계속 동작합니다.

* **댓글이 너무 많은 경우**
  시청자가 많아 댓글이 대량으로 흘러오는 방송에서는 AI가 모든 댓글에 반응하려고 해서 계속 말하게 될 수 있습니다.
