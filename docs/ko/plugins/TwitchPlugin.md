# 🎮 Twitch 라이브 연동 (TwitchPlugin.py)

Twitch 연동 플러그인입니다.  
스트리머의 음성 지시로 AI가 Twitch의 다양한 작업을 자동 수행합니다.

<img src="../../../images/TwitchPlugin.png" width="600">

---

## 🌟 주요 기능

| 기능 | 설명 | 계정 제한 |
|---|---|---|
| 채팅 읽기 | 시청자 댓글을 AI에 실시간 주입 | 없음 |
| 채팅 쓰기 | AI가 Twitch 채팅에 작성 | 없음 |
| 설문(Poll) | 시청자 투표 생성 및 집계 | **제휴/파트너만** |
| 예측(Prediction) | 시청자가 채널 포인트 베팅 | **제휴/파트너만** |
| 제목 변경 | 음성으로 방송 제목 변경 | 없음 |
| 클립 생성 | 직전 약 30초 클립 | 없음 |
| 시청자 수 | 동시 시청자 수 확인 | 없음 |
| 봇 제외 | 특정 사용자 AI 주입 제외 | - |

## ⚙️ 연동 방법

### Step 1: 접속 모드 선택

#### IRC 모드 (모든 방송, 채팅 읽기만)

**인증 불필요.** 모든 Twitch 채널의 댓글을 읽을 수 있습니다.

> Step 2-3을 건너뛰고 Step 4로 이동하세요.

#### 인증 모드 (내 방송, 전체 기능)

**OAuth2 인증 필요.** 자신의 채널에 연결하여 전체 기능을 사용합니다.

---

### Step 2: Twitch Developer Console에서 앱 등록

1. [Twitch Developer Console](https://dev.twitch.tv/console/apps)에 로그인
2. "애플리케이션 등록" 클릭
3. Client ID를 복사

4. **Client Secret (권장):**  
   "새 시크릿" 버튼으로 생성하여 복사합니다.

   > ⚠️ Client Secret이 없으면 약 4시간 후 토큰이 만료되어 재인증이 필요합니다.

### Step 3: TeloPon에서 인증

1. TeloPon → 확장 기능 → "Twitch 라이브 연동" 톱니바퀴 클릭
2. Client ID와 Client Secret을 입력
3. "Twitch로 인증" 버튼 클릭
4. 브라우저에서 인증 완료

---

### Step 4: 연결 후 라이브 시작

1. **인증 모드:** "연결" 버튼 클릭
2. **IRC 모드:** 채널명 입력 후 "연결" 클릭
3. TeloPon 메인 화면에서 **"🔴 라이브 시작"** 클릭

> **중요:** 라이브 시작 **전에** "연결"을 먼저 누르세요.

## 🎙️ 발화 예시

| 스트리머 발화 | AI 동작 |
|---|---|
| "채팅에 '안녕하세요' 써줘" | Twitch 채팅에 투고 |
| "좋아하는 색 설문해줘" | 60초 설문 생성 |
| "집계해줘" | 설문 마감 후 결과 발표 |
| "이 보스 잡을 수 있을지 베팅시켜" | 예측 생성 |
| "정답은 1번!" | 예측 확정 |
| "제목 변경해줘, 잡담 방송으로" | 방송 제목 변경 |
| "클립해줘!" | 직전 약 30초 클립 생성 |
| "지금 몇 명?" | 시청자 수 확인 |
