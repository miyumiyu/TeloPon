# 💬 Discord 실시간 연동 (discord_integration.py)

이 플러그인을 사용하면 TeloPon의 AI가 Discord 서버의 지정된 채널에서 오는 댓글을 실시간으로 읽고 반응합니다!

설정은 세 가지 주요 단계로 구성됩니다: **"① 플러그인 설치"**, **"② Discord에서 Bot 생성 (토큰 취득)"**, **"③ TeloPon에서 설정"**. 순서대로 따라 하면 초보자도 약 5분 안에 완료할 수 있습니다!

---

## 📥 Step 1: 플러그인 설치

이 확장은 TeloPon 기본 패키지에 포함되어 있지 않으므로 먼저 TeloPon에 추가합니다.

1. 파일 **`discord_integration.py`**를
   🔗 [TeloPon 공식 확장 플러그인 팩 v1.0 (Discord & Slack)](https://github.com/miyumiyu/TeloPon/releases/tag/plugins-v1.0)에서 다운로드합니다.
2. PC에서 `TeloPon-XXX` 폴더(메인 애플리케이션 폴더)를 엽니다.
3. 다운로드한 `discord_integration.py` 파일을 그 안의 **`plugins`** 폴더에 직접 배치합니다.
4. TeloPon을 실행(또는 재시작)합니다. 메인 화면의 "🔌 확장" 목록에 **"💬 Discord 실시간 연동"**이 표시되면 설치 성공!

---

## 🛠️ Step 2: Discord Bot 생성 및 "토큰" 취득

AI가 Discord 댓글을 읽으려면 자신만의 전용 "Bot(로봇)"을 만들어야 합니다.

### 1. 개발자 포털 접속
PC 브라우저에서 [Discord Developer Portal](https://discord.com/developers/applications)에 접속하여 평소 사용하는 Discord 계정으로 로그인합니다.

### 2. 애플리케이션(Bot) 생성
1. 우상단의 **"New Application"** 버튼을 클릭합니다.
2. `Name`에 원하는 이름(예: `TeloPon Bot`)을 입력하고, 약관 체크박스를 체크한 후 **"Create"**를 누릅니다.

### 3. [필수] 메시지 읽기 권한 켜기
**※ 이것 없이는 Bot이 댓글을 읽을 수 없습니다! 필수입니다.**
1. 왼쪽 메뉴에서 **"Bot"**을 클릭합니다.
2. 약간 아래로 스크롤하여 **"Privileged Gateway Intents"** 섹션을 찾습니다.
3. **"Message Content Intent"** 스위치를 **ON(녹색)**으로 켜고, 팝업이 나타나면 "Save Changes"를 누릅니다.

### 4. 토큰(키) 복사
1. 같은 **"Bot"** 화면의 상단으로 돌아갑니다.
2. "Token" 필드 옆의 **"Reset Token"** 버튼을 누르고 "Yes, do it!"을 선택합니다.
3. 긴 문자열(이것이 토큰)이 나타납니다 — **"Copy"**를 눌러 복사합니다.
*(※ 화면을 닫으면 다시 표시되지 않으므로 메모장 등에 붙여넣으세요.)*

---

## 🖥️ Step 3: TeloPon에서 설정 및 연결

TeloPon 메인 화면에서 "🔌 확장"을 열고 **"💬 Discord 실시간 연동"**의 설정(⚙️) 버튼을 눌러 패널을 엽니다.

![Discord 연동 설정 패널](../../../images/slack_integration.png)

### 1. 토큰 입력
* Step 2에서 복사한 긴 문자열(토큰)을 "Bot Token" 입력 필드에 붙여넣습니다.

### 2. Bot을 서버에 초대
* 토큰을 입력한 상태에서 바로 아래의 **"🌐 이 Bot을 서버에 초대"** 버튼을 클릭합니다.
* 브라우저가 자동으로 열리며 Discord 초대 화면이 표시됩니다.
* Bot을 추가하려는 서버를 선택하고 "예" → "승인"을 통해 진행합니다.
*(※ Discord에서 채널에 "○○가 TeloPon Bot을 추가했습니다"가 표시되면 성공입니다.)*

### 3. 채널 목록 가져오기
* TeloPon 화면으로 돌아가 **"🔄 토큰 확인 & 채널 목록 가져오기"** 버튼을 누릅니다.
* 성공하면 "✅ 성공: ○개 채널 가져오기 완료!"가 표시됩니다.

### 4. 모니터링할 채널 선택
* 아래의 "모니터링할 채널" 드롭다운(▼)을 엽니다 — 서버의 채널(💬 텍스트, 🔊 음성 등)이 나열됩니다.
* AI가 읽길 원하는 채널을 선택합니다.

### 5. 연결!
* 마지막으로 파란색 **"시작(연결)"** 버튼을 누릅니다!
* 상태가 녹색 "⚡ 상태: 연결됨"으로 바뀌면 준비 완료입니다.

---

## 💡 팁 및 사용 시 주의 사항

설정 완료 후에는 평소처럼 TeloPon 메인 화면에서 "라이브 연결 시작(AI 실행)"을 누르기만 하면 됩니다!
지정한 Discord 채널에 글이 올라오면 약 0.1초의 지연으로 AI에게 댓글이 전송되고 AI가 반응합니다.

* **프롬프트 커스터마이즈**: 설정 패널 하단의 "AI에게 전달하는 지시 텍스트"를 수정하면 AI의 반응 방식을 변경할 수 있습니다. (예: "이것들은 Discord 청취자들의 댓글입니다. 친근하게 답해주세요!")
* **Bot이 오프라인이 되었나요?**: Discord Bot은 TeloPon의 "연결" 버튼이 활성화된 동안에만 "온라인(녹색 점)"으로 표시됩니다. TeloPon을 닫으면 자동으로 오프라인이 됩니다.
