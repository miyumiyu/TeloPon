# 🔑 초보자 가이드: 무료 API 키 발급받기!

TeloPon을 실행하려면 Google에서 제공하는 AI 키(API 키)가 필요합니다.
**신용카드 없이도 완전 무료 티어를 이용할 수 있으며**, 개인 스트리밍 및 취미 용도로는 충분합니다!

처음이더라도 PC나 스마트폰에서 아래 단계를 순서대로 따르면 쉽게 발급받을 수 있습니다.

---

### Step 1: API 키 생성 페이지 열기
먼저 Google AI Studio로 이동하여 로그인하고, 화면의 **"Create API key"** 버튼을 클릭합니다.

👉 **[Google AI Studio (API 키 페이지)](https://aistudio.google.com/api-keys)**

<img src="../../images/api-key1.png" width="300">

### Step 2: 새 프로젝트 만들기
"Select an imported project"를 클릭한 후, 메뉴에서 **"+ Create project"**를 선택합니다.

<img src="../../images/api-key2.png" width="300">

### Step 3: 프로젝트 이름 지정
"Create new project" 화면이 나타납니다. "Name your project" 필드에 알기 쉬운 이름(예: **`TeloPon`**)을 입력한 후 "Create project"를 누릅니다.

<img src="../../images/api-key3.png" width="300">

### Step 4: API 키 이름 지정
다음으로 "Create a new key" 화면이 나타납니다. "Key name" 필드에 이름(예: **`TeloPon API Key`**)을 입력한 후 "Create key"를 누릅니다.

<img src="../../images/api-key4.png" width="300">

### Step 5: API 키 복사
생성이 완료되면 API 키 목록으로 돌아갑니다. 새 키가 표시됩니다(예: `...3eb8`). 클릭하면 전체 문자열(`AIza...`로 시작)을 복사할 수 있습니다.

<img src="../../images/api-key5.png" width="300">
