> 👉 **[TeloPon v2.30b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.30b/TeloPon-230b.zip)**

## ✨ 새로운 기능

### 🎲 TRPG GM 기능 — AI가 게임 마스터가 된다, 시청자와 함께

판타지 TRPG・크툴루 신화 TRPG 두 종류의 프롬프트를 추가. AI가 GM/KP가 되어, 방송자와 시청자를 끌어들이는 **시청자 참여형 세션**을 진행할 수 있습니다.

- **판타지 TRPG 프롬프트**: 전투・협상・탐색을 망라한 정통 시나리오
- **크툴루 신화 TRPG (CoC) 프롬프트**: SAN값・광기・1d100 판정 시스템, 호러 연출
- **AI가 정확하게 주사위를 굴린다**: 결과 텔롭이 자동 생성 (출목・보정값・성공/실패)
- **🗳️ 시청자가 앙케트로 이야기에 개입**: YouTube / 니코니코 / Twitch 모든 플랫폼에서 사용 가능. 시나리오의 분기점("왼쪽 문? 오른쪽 문?", "싸운다? 도망친다?")에서 시청자가 운명을 결정
- **💬 시청자 댓글이 이야기에 스며든다**: 댓글을 "속삭임", "주막의 소문", "꿈의 계시"로 작중에 등장시켜 이야기의 소재로 받아들임

### 🎲 음성 주사위 굴리기 — 일반 방송에서도 부담 없이 사용

TRPG 프롬프트 외에도, 방송자의 음성으로 AI에게 주사위를 굴리게 할 수 있습니다.

- "**2D6+3 굴려줘**", "**1D6 주사위 굴려**", "**주사위 굴려줘**"라고 말하기만 하면 됨
- 결과 텔롭에 표기・출목・합계가 표시됨 (예: 🎲 2d6+3, 출목 4 5, 합계 12)
- 일반 방송에서는 심플 표시, TRPG 문맥에서는 DC・성공/실패 표시로 자동 전환

### 💾 스토리 저널 기능 — 이야기가 사라지지 않는다

긴 콘텐츠의 "현재 상태"를 AI가 영구 저장. 리로드・재접속・방송일 넘김에서도 이어서 재개할 수 있습니다.

- 방송자가 "**지금 상황 저장해둬**", "**여기까지 세이브해**"라고 말하면 AI가 자동으로 저장
- 세션 시작 시 "**전회의 이어서 인계**" UI 토글로 이전 저널을 자동 주입
- **프롬프트 단위로 관리**: 판타지 GM・CoC・일반 방송 등 프롬프트마다 별도의 저널로 저장 (섞이지 않음)
- TRPG뿐만 아니라, **스토리 게임 방송・연재 기획・캐릭터 역할 분담** 등 모든 장기 콘텐츠에 사용 가능
- "새로운 이야기로", "리셋"이라고 말하면 아카이브 후 신규 시작

### 🪪 신규 윈도우 스타일 window-status — 사라지지 않는 텔롭 프레임

**지시가 있을 때까지 사라지지 않는 상시 표시 윈도우**를 신규 추가했습니다. 재전송하면 같은 프레임의 내용이 덮어쓰기 갱신됩니다.

- **OBS의 브라우저 소스에서 조작 가능**: 드래그로 이동・휠로 확대축소・더블 클릭으로 일시 숨김
- 방송자가 "스테이터스 닫아"라고 말하면 AI가 제거

응용 예: TRPG 스테이터스 (HP/SAN/MP), 첫 방문자 카운터, 방송 시간, 시청자 수, 목표 달성까지의 수치… 상시 보이고 싶은 정보라면 무엇이든.

### ✨ 첫 방문자 카운터 플러그인 (확장)

댓글에서 "처음", "첨", "first time" 등의 키워드를 감지해 누적 인원을 window-status에 상시 표시하는 플러그인 (위 응용 예의 구현 버전).

- 같은 시청자는 1회만 카운트 (중복 방지)
- 키워드・헤더 문자를 UI에서 편집 가능
- YouTube / Twitch / 니코니코 등 댓글 수신 플러그인 전체에서 자동 집약
- 영속화 대응 (방송을 넘어 카운트 지속, 리셋도 가능)
