## 🆕 신규 기능

### ⏰ 타이머·알람 기능 (Function Calling)
스트리머의 음성 지시만으로 AI가 타이머나 알람을 설정하고, 발동 시 자동으로 알림을 띄웁니다.

- **"5분 타이머 걸어줘"** → 5분 후 텔롭으로 알림
- **"17시에 알려줘"** → 17:00에 텔롭으로 알림
- **"타이머 취소해"** → 설정 중인 타이머 취소

또한 **액션 포함 지시**에도 대응합니다.

- **"3분 후에 설문을 종료해"** → 3분 후 AI가 자동으로 설문 종료 명령을 발행
- **"5분 후에 제목을 바꿔"** → 5분 후 AI가 자동으로 제목 변경 명령을 발행

상대 시간(초/분/시)과 절대 시각(HH:MM) 모두 지원합니다. 방송 진행을 AI에게 맡길 수 있게 되었습니다.

### 🔄 라이브 중 리로드 기능
라이브 중에 AI의 기억(컨텍스트)을 통째로 리셋할 수 있는 "리로드 버튼"을 추가했습니다.

- **대기 텔롭을 표시하지 않고** 방송 중단 없이 세션을 새로 시작
- 플러그인·마이크 모니터·타이머는 그대로 동작 지속
- AI가 잘못된 정보나 오인식을 끌고 다니기 시작했을 때 "재정비"에 유효

> **이슈 [#13](https://github.com/miyumiyu/TeloPon/issues/13) (마이크 뮤트 기능 요청)에 대한 대응**: 수동 뮤트로는 타이밍을 놓칠 수 있으므로, 오염된 컨텍스트를 사후에 리셋하는 방식을 채택했습니다.

## 🧠 AI 응답 개선

### 시간 필러 억제
화제가 없을 때 AI가 자발적으로 "지금 ◯시 ◯분을 알려드립니다"라고 시간을 필러로 사용하는 문제를 수정했습니다. 질문받았을 때만 시간을 응답하도록 4개 언어 프롬프트를 강화했습니다.

## 🔌 확장 플러그인

### 🔊 Gemini TTS 플러그인 (신규)
최신 **Gemini 3.1 Flash TTS Preview**에 대응한 고품질 텔롭 읽기 플러그인을 신규 추가했습니다.

- 30가지 프리셋 보이스에서 선택
- 감정 태그(badge)·캐릭터 지시·속도·딜레이로 표현 제어
- TeloPon 본체의 Gemini API 키를 그대로 사용 (추가 인증 불필요)
- 출력 장치 선택 지원

> ⚠️ **주의**: Gemini TTS API는 레이트 리미트가 엄격하며, 무료 API로는 5회 정도에서 리미트에 도달합니다. **체험용**으로 사용해 주세요.
