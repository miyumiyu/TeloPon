## ✨ 주요 추가·개선

### 🆕 신기능: 자동 끼어들기 모드

마지막 텔롭이 표시된 후 일정 시간이 경과하면, AI가 지금까지의 화제를 자동으로 요약·정리하는 텔롭을 출력해 주는 기능입니다.

* UI의 "발화 모드" 콤보박스에서 **"자동 끼어들기"** 를 선택하여 활성화
* "개입 시간(초)" 슬라이더로 발동 간격을 설정
* 콤보박스 옆에 **발동까지의 카운트다운**이 실시간으로 표시됩니다

**이런 상황에 최적:**
뉴스·정보 방송이나 긴 잡담 세션에서 AI가 정기적으로 토픽을 정리해 주므로 시청자도 대화 흐름을 따라가기 쉬워집니다. 개입 시간은 60〜120초를 권장합니다.

---

### 🔄 UI 개선: 발화 모드를 콤보박스로 선택

기존의 체크박스 방식 "자동 개입 ON/OFF"에서 **콤보박스로 발화 모드를 선택하는 방식**으로 변경했습니다.

| 모드 | 동작 |
|---|---|
| **None** | 자동 개입 없음 |
| **자동 발화** | 화자가 침묵한 후 N초 뒤에 AI가 자발적으로 화제를 꺼냄 (기존 기능) |
| **자동 끼어들기** | 마지막 텔롭 출력 후 N초 뒤에 AI가 화제를 요약·정리함 (신기능) |

* 콤보박스 옆 카운트다운으로 **발동까지 남은 시간**을 한눈에 확인할 수 있습니다
* AI 처리 중에도 카운트다운이 멈추지 않아 정확한 발동 타이밍을 파악할 수 있습니다
* 라이브 접속 중에도 모드를 전환할 수 있습니다
