# TeloPon V2.16b 릴리스 노트

## 🎮 Twitch 연동 플러그인 대폭 강화

OAuth2 인증을 지원하여, 댓글 읽기만 가능했던 Twitch 연동이 전면 진화했습니다.

- **댓글 작성** - "○○라고 채팅에 써줘"로 AI가 Twitch 채팅에 투고
- **설문** - "좋아하는 색 설문해줘"로 투표 생성, "집계해줘"로 결과 발표
- **예측(베팅)** - "이 보스 잡을 수 있을지 베팅시켜"로 시청자가 포인트 베팅 (제휴/파트너)
- **클립 생성** - "지금 재밌었어! 클립해줘"로 직전 30초 저장
- **제목 변경** - "제목을 잡담 방송으로 변경해줘"로 즉시 변경
- **시청자 수** - "지금 몇 명 보고 있어?"로 AI가 알려줌
- **봇 제외** - nightbot 등을 더블클릭으로 제외 목록에 추가
- **인증 모드 / IRC 모드** - 전체 기능 또는 채팅 읽기만(인증 불필요) 전환 가능

## ▶️ 확장 플러그인에 YouTube OAuth 연동 추가

YouTube Data API + OAuth2를 활용한 고급 YouTube 연동 플러그인을 출시했습니다.

- **댓글 작성** - "○○라고 댓글 달아줘"로 AI가 YouTube 채팅에 투고
- **설문** - "설문해줘"로 YouTube 공식 설문 생성, "집계해줘"로 결과를 AI가 발표
- **제목 변경** - "제목을 ○○로 바꿔줘"로 방송 제목 변경
- **시청자 수** - "지금 몇 명?"으로 동시 시청자 수를 AI가 알려줌
- **방송 선택** - 예약된 방송 목록에서 선택하거나 URL로 직접 지정
- **권한 자동 판별** - 소유자/관리자 여부를 자동 판별하여 사용 가능한 기능 표시

스크린샷이 포함된 GCP 설정 가이드도 준비했습니다.

👉 [TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)

## 🛡️ Twitch / YouTube OAuth 연동에 명령 사용 권한 추가

Twitch 연동 및 YouTube OAuth 연동의 각 기능에 "시청자 가능" 체크박스를 추가했습니다.  
시청자 댓글에서 명령을 실행할 수 있는지 여부를 기능별로 개별 제어할 수 있습니다.  
예를 들어, 제목 변경이나 설문은 스트리머의 음성으로만 실행하고, 시청자 댓글에서는 실행하지 않도록 설정할 수 있습니다.

## 🎤 설명 창 · 이름표를 음성으로 닫기

"설명 닫아줘", "이름표 숨겨줘"라고 말하면 AI가 화면의 창을 닫습니다.
