# 🌐 Release V1.20b：4개 언어 지원 & Twitch 연동 플러그인 기본 포함!

이번 업데이트는 **글로벌 대응**과 **방송 플랫폼 확장**을 메인으로 한 버전입니다.
UI와 문서를 한 번에 4개 언어로 대응하여, 해외 방송인에게도 사용하기 쉬운 어시스턴트 툴로 진화했습니다.
또한 Twitch 채팅 읽기와 방송 정보 연동이 기본 플러그인으로 포함되어, YouTube뿐만 아니라 Twitch에서의 방송도 바로 시작할 수 있습니다!

---

## ✨ 주요 신기능

### 🌐 UI 4개 언어 지원 (日本語 / English / 한국어 / Русский)

TeloPon 본체 UI (버튼·상태·메뉴 등)가 OS의 언어 설정에 따라 자동으로 전환됩니다.

* **일본어**·**영어**·**한국어**·**러시아어** 지원.
* 시작 시 OS 언어를 감지하며, 미지원 언어의 경우 영어로 폴백됩니다.
* 플러그인 UI 표시도 동일하게 다국어 대응되어 있습니다.

---

### 🎮 Twitch 연동 플러그인 기본 포함 (TwitchPlugin.py)

Twitch 방송을 지원하는 새 플러그인이 별도 설치 없이 사용 가능한 **기본 번들 플러그인**으로 추가되었습니다.

* **채팅 읽기**: 채널명을 입력하기만 하면 시청자의 댓글을 실시간으로 가져와 AI에 전달합니다. **Client ID 없이도 동작**합니다.
* **스트림 정보 주입**: Twitch Developer Console에서 Client ID / Client Secret을 설정하면 방송의 "제목", "카테고리", "썸네일"도 AI에 자동 전송됩니다.
* **URL / 채널명 모두 입력 가능**: `https://twitch.tv/mychannel` 형식과 `mychannel` 단독 모두 인식합니다.

👉 [Twitch 연동 도구 - 상세 사용 가이드](../plugins/TwitchPlugin.md)

## 💡 방송인을 위한 한 가지 팁

**Twitch와 YouTube를 동시에 방송하시는 분께**
Twitch 플러그인과 YouTube 플러그인은 동시에 활성화할 수 있습니다. 양쪽 채팅을 동시에 AI에 전달하여, 멀티 플랫폼 방송에서도 모든 시청자의 댓글에 AI가 반응하는 "전원 참여형" 방송 체험을 실현할 수 있습니다!

**Twitch 플러그인은 바로 시도할 수 있습니다!**
Client ID 취득은 생략 가능합니다. 채널명을 입력하고 "연결" → "🔴 라이브 연결 시작"만으로 Twitch 댓글 연동을 바로 시도할 수 있습니다.
