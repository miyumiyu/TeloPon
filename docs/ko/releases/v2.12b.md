# 🚀 Release V2.12b：reply 카드 디자인 개편·explain 위치 기억 지원!

> 👉 **[TeloPon v2.12b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

이번 업데이트에서는 **`window-reply`를 SNS 카드 스타일로 리디자인**하고, 아바타 표시·슬라이드 애니메이션·글로우 이펙트를 추가했습니다. 또한 설명 창（`window-explain`）의 위치와 스케일이 OBS 재시작 후에도 유지됩니다.

---

## ✨ 주요 추가 및 개선 사항

### 🆕 window-reply SNS 카드 디자인

`window-reply` 텔롭이 SNS 리플 카드 스타일로 리디자인되었습니다.

* **원형 아바타 아이콘** 표시（YouTube 등 프로필 이미지 지원）
* 아바타 우측 하단에 **↩ 답글 배지** 표시
* 텔롭 등장 시 **슬라이드 인 애니메이션**（앵커 위치에 따라 방향 자동 판정）
* 다크 배경 ＋ **글로우（광채） 이펙트**（전 4테마 대응：default / comic / honwaka / tsundere）

---

### 🆕 explain 위치·스케일 영속화

`window-explain`（설명 창）의 위치와 줌 배율이 `localStorage`에 저장됩니다.

* 드래그로 이동한 위치가 **OBS 재시작 후에도 유지**됩니다
* 휠로 줌한 스케일도 **보존**됩니다
* **더블클릭으로 위치·스케일 초기화** 가능

---

### 🔧 기타 수정

* `window-reply` 슬라이드 애니메이션 후 저장된 스케일이 적용되지 않던 문제 수정
* 더블클릭 리셋 시 텔롭이 순간 좌상단으로 이동하는 문제 수정
* OBS 브라우저 소스가 CSS를 계속 캐시하는 문제 수정（서버 no-cache 헤더 ＋ URL 타임스탬프）
