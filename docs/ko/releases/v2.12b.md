# 🚀 Release V2.12b：텔롭 UI/UX 전면 개선·사고 레벨 설정 지원!

> 👉 **[TeloPon v2.12b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

이번 업데이트에서는 텔롭의 비주얼과 조작성을 전면적으로 개선했습니다. 또한 Gemini 3.1에 대응한 사고 레벨 설정이 가능해졌습니다.

---

## ✨ 텔롭 UI/UX 전면 개선

### 🆕 window-reply SNS 카드 디자인

`window-reply` 텔롭이 SNS 리플 카드 스타일로 리디자인되었습니다.

* **원형 아바타 아이콘** 표시（YouTube 등 프로필 이미지 지원）
* 아바타 우측 하단에 **↩ 답글 배지** 표시
* 텔롭 등장 시 **슬라이드 인 애니메이션**（앵커 위치에 따라 방향 자동 판정）
* 다크 배경 ＋ **글로우（광채） 이펙트**（전 4테마 대응：default / comic / honwaka / tsundere）

---

### 🆕 애니메이션 부드러움 개선

드래그 이동, 텔롭 등장·소멸 애니메이션을 전면적으로 개선했습니다.

* 등장：**scale(0)에서 부드럽게 확대**하여 나타남
* 소멸：**scale(0)까지 부드럽게 축소**하여 사라짐（기존의 갑자기 사라지는 동작 수정）
* 드래그 이동：**지연이나 끊김 없는 즉각적인 추종**
* 큐 전환：다음 아이템이 있을 경우 **페이드아웃과 페이드인이 오버랩**되는 매끄러운 전환

---

### 🆕 explain「다음」배지 / nameplate「×」닫기 버튼

텔롭 조작성을 향상하는 버튼을 추가했습니다.

* **explain 프레임**：큐에 아이템이 있을 때 우측 하단에 큐 수 ＋「›」배지 표시. 클릭으로 즉시 다음 아이템으로 전환
* **nameplate 프레임**：우측 상단에「✕」닫기 버튼 상시 표시. 클릭으로 즉시 삭제

---

### 🆕 explain 드래그 개선

`window-explain`（설명 창）의 드래그 조작을 대폭 개선했습니다.

* 드래그로 이동한 위치가 **OBS 재시작 후에도 유지**됩니다
* 휠로 줌한 스케일도 **보존**됩니다
* **더블클릭으로 위치·스케일 초기화** 가능
* 뷰포트 끝으로 밀면 프레임이 **수평 방향으로 압축**, 되돌리면 확대（좌우 40px 고정 마진）
* **드래그 중 타이머 일시 정지**（드래그 중 텔롭이 사라지지 않음）

👉 **[OBS에서 텔롭 조작에 대한 자세한 설명은 여기를 참조하세요](https://github.com/miyumiyu/TeloPon/blob/main/docs/ko/05_obs_control.md)**

## ✨ 사고 레벨 설정

### 🆕 Gemini 3.1 사고 레벨 지원

Gemini 3.1의 새로운 사고 기능에 대응했습니다. UI의「💭 사고 레벨」드롭다운에서 설정할 수 있습니다.

| 레벨 | 설명 |
|---|---|
| **minimal** | 최저 레이턴시 우선（기본값） |
| **low** | 간단한 작업용 |
| **medium** | 중간 정도의 복잡도용 |
| **high** | 복잡한 추론·다단계 플래닝용 |

* 기존의 `thinking_budget`（토큰 수 지정）은 폐지되었습니다
* 설정은 `settings.json`에 저장되어 다음 실행 시에도 복원됩니다
