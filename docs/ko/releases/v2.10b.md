# 🚀 Release V2.10b：마작 패 표시·시청자 아바타 지원·안정성 강화!

> 👉 **[TeloPon v2.10b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.10b/TeloPon-210b.zip)**

이번 업데이트는 **Gemini 3.1 Flash Live의 추가 안정화·최적화**를 중심으로, 마작 방송 지원·시청자 아바타 표시 등 다양한 기능을 추가했습니다.

---

## ✨ 주요 추가·개선

### ⚡ Gemini 3.1 Flash Live 최적화·안정화

Gemini 3.1 Flash Live용 연결 처리를 전면 리팩터링했습니다.

* 텔롭 응답 속도가 한층 향상되어 더욱 실시간에 가까운 표시가 가능해졌습니다
* 세션 재연결 안정성 개선

---

### 🆕 시청자 댓글 답변 강화（window-reply）

댓글 답변 텔롭 `window-reply`의 디자인과 기능을 대폭 강화했습니다.

* **말풍선 스타일**로 변경하여 화면 **좌측 상단**에 배치
* **YouTube·Twitch** 댓글 작성자 아바타（썸네일）를 답변 텔롭에 표시
  * YouTube는 댓글 수신 시 자동으로 아바타를 가져옵니다
  * Twitch는 Helix API를 사용합니다（Client ID·Secret ID를 설정한 경우에만）→ [Twitch 플러그인 설정](../plugins/TwitchPlugin.md)

---

### 🆕 마작 방송 지원

마작 방송을 위한 새로운 기능을 추가했습니다.

#### 마작 패 인라인 표시 태그

프롬프트 내에 패 태그를 작성하면 텔롭에 패 이미지가 자동으로 표시됩니다.

| 종류 | 태그 |
|---|---|
| 만쯔 | `<M1>` 〜 `<M9>` |
| 소쯔 | `<S1>` 〜 `<S9>` |
| 핀쯔 | `<P1>` 〜 `<P9>` |
| 자패 | `<J1>`동 `<J2>`남 `<J3>`서 `<J4>`북 `<J5>`백 `<J6>`발 `<J7>`중 |

#### 마작 어드바이저 프롬프트 `r_mahjong.txt`

* OBS 캡처와 조합하여 패 버림 어드바이스를 텔롭으로 표시할 수 있습니다
* 화면에 확실하게 보이는 패만을 근거로 어드바이스하도록 설계되어 있습니다
* ⚠️ **AI가 이미지를 잘못 읽어 존재하지 않는 패를 언급하는 경우가 있습니다.** 엔터테인먼트로 즐겨주세요!
