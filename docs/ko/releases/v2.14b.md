# 🚀 Release V2.14b：플러그인 소결합화·텔롭 가시성 개선·VOICEVOX 읽기 추가!

> 👉 **[TeloPon v2.14b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.14b/TeloPon-214b.zip)**

이번 업데이트에서는 플러그인 아키텍처의 소결합화, 텔롭 가시성 개선, 확장 플러그인에 VOICEVOX 읽기를 추가했습니다.

---

## ✨ 플러그인 아키텍처 소결합화

### 🆕 fire-and-forget 방식

텔롭 출력 훅（`on_telop_output`）을 **fire-and-forget（발사 후 망각）** 방식으로 변경했습니다.

* Core는 플러그인 처리 완료를 **일절 기다리지 않습니다**
* 플러그인 내 오류나 지연이 **시스템 전체에 영향을 주지 않습니다**
* 딜레이 제어는 `set_telop_delay()` / `get_telop_delay()`로 분리

서드파티 플러그인이 안전하게 동작하는 기반이 마련되었습니다.

👉 **[플러그인 개발 가이드（상세）](https://github.com/miyumiyu/TeloPon/blob/main/docs/ko/03_plugin_dev.md)**

---

## ✨ 텔롭 가시성 개선

일반 텔롭의 스택 표시 시 투명도를 변경하여 오래된 텔롭의 가시성을 향상했습니다.

| 순번（최신→오래된） | 변경 전 | 변경 후 |
|---|---|---|
| 1번째（최신） | 100% | 100% |
| 2번째 | 50% | **75%** |
| 3번째 | 30% | **50%** |
| 4번째 이후 | 10% | **25%** |

---

## ✨ 확장 플러그인에 VOICEVOX 읽기 추가

### 🆕 VOICEVOX 읽기 플러그인

AI가 출력한 텔롭 텍스트를 **VOICEVOX** 음성 합성 엔진으로 실시간 읽어주는 확장 플러그인을 추가했습니다.

* **캐릭터 & 스타일 선택**: VOICEVOX 전체 캐릭터 + 성격（노멀/상냥/츤츤 등）을 드롭다운으로 선택
* **아이콘 표시**: 선택한 캐릭터·스타일의 아이콘이 설정 화면에 표시
* **5가지 음성 파라미터**: 속도·음량·음높이·억양·발화 전 무음을 실시간 조정
* **TOPIC 읽기**: ON이면 텔롭 상단 텍스트도 함께 읽기
* **텔롭 표시 제어**: 읽기만（텔롭 비표시）또는 딜레이 표시 가능
* **자동 연결 확인**: VOICEVOX 연결을 자동 확인. 연결 불가 시 UI 비활성화

👉 **[VOICEVOX 읽기 플러그인 상세 가이드](https://github.com/miyumiyu/TeloPon/blob/main/docs/ko/plugins/voicevox_plugin.md)**

> 💡 별도로 [VOICEVOX](https://voicevox.hiroshiba.jp/) 설치가 필요합니다.
