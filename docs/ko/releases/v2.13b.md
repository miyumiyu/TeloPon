# 🚀 Release V2.13b：플러그인 텔롭 출력 훅·TelopViewer 플러그인 추가!

> 👉 **[TeloPon v2.13b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.13b/TeloPon-213b.zip)**

이번 업데이트에서는 플러그인 기구에 텔롭 출력 훅을 추가하여 AI의 출력을 플러그인에서 수신할 수 있게 되었습니다. 또한 기존 텔롭 모니터를 폐지하고 동등한 기능을 플러그인으로 재구축했습니다.

---

## ✨ 플러그인에 텔롭 출력 훅 기구 추가

### 🆕 on_telop_output 훅

AI가 텔롭을 출력할 때마다 플러그인이 데이터를 실시간으로 수신할 수 있는 `on_telop_output` 훅을 신설했습니다.

* 플러그인은 **topic / main / window / layout / badge** 전체 데이터를 수신
* `<b1>` `<b2>` 태그와 마작 패 태그（`<P1>` 등）도 원본 데이터로 전달
* 반환값으로 OBS 텔롭 표시를 제어 가능

| 반환값 | 동작 |
|---|---|
| `0` | 즉시 표시（기본값） |
| `1` 이상 | N초 후 표시（딜레이） |
| `-1` | OBS 표시 억제（플러그인에는 데이터 전달） |

### 🛡️ 안전 기구

* 각 플러그인의 `on_telop_output`은 **타임아웃 1초**로 실행
* **3회 타임아웃** 시 자동으로 훅 호출 대상에서 제외

👉 **[플러그인 개발 가이드（on_telop_output 상세）](https://github.com/miyumiyu/TeloPon/blob/main/docs/ko/03_plugin_dev.md)**

## ✨ 텔롭 모니터 폐지 → TelopViewer 플러그인으로 대체

### 🆕 TelopViewer 플러그인

기존의 「텔롭 모니터」를 폐지하고 `on_telop_output` 훅을 활용한 **TelopViewer 플러그인**으로 재구축했습니다.

* AI가 출력한 텔롭 내용을 **조작 패널에서 실시간 표시**
* **B1/B2 태그 색상 구분 표시**（B1=빨강계, B2=시안계）
* **표시 딜레이를 드롭다운으로 설정** 가능（0/1/2/3/5/10/-1）
* 플러그인 목록의「조작 패널」버튼에서 열기만 하면 사용 가능
