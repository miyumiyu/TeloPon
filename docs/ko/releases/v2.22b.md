## 🎮 OBS 연동 개선

### 리플레이 버퍼 저장 지원
"리플레이 저장해줘", "지금 거 남겨줘"라고 말하면 OBS의 리플레이 버퍼를 저장합니다.  
사전에 OBS에서 리플레이 버퍼를 활성화해 둘 필요가 있습니다.

### UI/UX 개선
- OBS WebSocket **연결 버튼**을 추가하여 연결 상태를 알기 쉽게 표시
- 연결 성공 시 씬 목록을 자동 취득하고 캡처 프리뷰를 표시
- 연결 실패 시 캡처, 녹화, 리플레이 버퍼를 전부 그레이아웃 처리
