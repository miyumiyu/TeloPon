# 🚀 Release V2.15b：AI 아이콘·UI 재구성·안정성 향상!

> 👉 **[TeloPon v2.15b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.15b/TeloPon-215b.zip)**

이번 업데이트에서는 텔롭에 AI 아이콘 표시 기능 추가, UI 재구성, 통신 안정성 향상을 실시했습니다.

---

## ✨ AI 아이콘 기능

### 🆕 텔롭에 AI 아이콘 표시

AI의 성격에 맞는 아이콘 이미지를 텔롭 왼쪽에 표시할 수 있게 되었습니다.

* UI의 「AI 아이콘 선택」 버튼에서 이미지 설정
* **프레임 높이에 맞춰 자동 리사이즈**（정사각형·중앙 트리밍）
* 이미지가 없으면 기존과 동일한 표시
* 대상: variety / news / digital / cute / simple
* reply / explain / nameplate에는 표시되지 않음

---

## ✨ UI 재구성

설정 화면의 레이아웃을 재정리하여 더 알기 쉽게 했습니다.

* **⚙️ AI 설정**: API Key·인증 상태·모델·사고 레벨·AI 성격·AI 아이콘
* **📡 방송 설정**: 스트리머 이름·마이크 선택·미터·임계값·테마

---

## 🔧 안정성 향상

### AI 무응답 시 자동 복구 기구 추가

AI의 응답이 장시간 끊긴 경우 자동으로 세션을 재연결하는 기구를 추가했습니다.
방송 중 AI가 침묵하는 문제의 발생 빈도가 크게 줄어듭니다.

---

## 🔧 기타 수정

* v2.14b에서 일부 플러그인이 동작하지 않던 문제 수정（ctypes.wintypes를 빌드에 추가）
* 텔롭 표시 딜레이가 기동 시 이전 값을 유지하던 문제 수정（기동 시 항상 0으로 리셋）
