# 🎙️ TeloPon V2.20b — "목소리만으로 방송의 모든 것을 조작하다."

> 👉 **[TeloPon v2.20b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.20b/TeloPon-220b.zip)**

---

## 🌟 하이라이트

방송 플랫폼 API에 완벽 대응했습니다. YouTube Live, Twitch, 니코니코 생방송에서
투표 생성, 댓글 작성, 방송 제목 변경, 시청자 수 확인 등
**플랫폼 고유의 방송 기능을 AI에게 말하는 것만으로 조작**할 수 있게 되었습니다.

플러그인 매니저가 탑재되어 필요한 기능을 원클릭으로 추가할 수 있습니다.
확장 플러그인은 전용 리포지토리로 분리되었으며, 썸네일 표시부터 OBS 녹화, 트윗 게시까지
목소리 하나로 방송의 모든 것을 컨트롤하는 경험을 제공합니다.

---

## 🆕 새로운 기능

### 📺 방송 플랫폼 API 완벽 대응
- **YouTube Live**: 댓글 읽기/쓰기, 투표 생성/집계, 방송 제목 변경, 시청자 수 확인, 슈퍼챗 대응 (OAuth 연동)
- **Twitch**: 댓글 읽기/쓰기, 투표, 예측(베팅), 클립 생성, 제목 변경, 시청자 수 확인 (OAuth2 Device Code Flow)
- **니코니코 생방송**: 댓글 취득, 선물/니코니 광고 알림, 투표 생성/집계, 운영 댓글 게시, 시청자 수 확인 (동봉 플러그인으로 승격)

### 🔧 플러그인 매니저
- UI에서 플러그인 활성화/비활성화 전환
- 확장 플러그인을 원클릭으로 다운로드 & 바로 사용
- 직접 만든 플러그인의 수동 설치도 지원

### 🎤 썸네일 음성 조작
- "썸네일 보여줘", "크게 해줘", "지워줘"로 OBS에 썸네일 이미지를 표시/조작
- 방송 플랫폼 연결 시 자동으로 활성화

### 🎮 OBS 연동 강화
- OBS 녹화 음성 조작 ("녹화 시작해줘", "녹화 멈춰줘")
- 캡처 대상을 씬 목록에서 드롭다운 선택

### 🗣️ 시스템 음성 명령
- "설명 닫아줘", "네임플레이트 닫아줘"로 AI가 화면의 윈도우를 닫습니다

## 📦 확장 플러그인

확장 플러그인은 전용 리포지토리에서 관리됩니다.  
플러그인 매니저에서 원클릭으로 다운로드할 수 있습니다.

🔗 **[TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)**

| 플러그인 | 개요 |
|---|---|
| 📺 **YouTube Live+** | YouTube OAuth 연동 (댓글 읽기/쓰기, 투표, 제목 변경, 슈퍼챗) |
| 🐦 **X (Twitter)** | 해시태그 취득, 썸네일/화면 첨부 트윗 게시 |
| 🔊 **자막 읽기 (Windows)** | Windows 기본 음성으로 자동 읽기 |
| 🔊 **자막 읽기 (VOICEVOX)** | VOICEVOX 음성 합성 |
| 📊 **PowerPoint 조작** | 슬라이드쇼 음성 조작, 노트 자동 주입 |
| 💬 **Discord** | Discord 채널 댓글 취득 |
| 💬 **Slack** | Slack 채널 댓글 취득 |
| 🎮 **VCI 자막 전송** | VirtualCast VR 자막 |

## 🔄 개선 및 변경

- 플러그인명을 통일 명명 규칙으로 변경 (📺 방송 / 🐦 SNS / 💬 채팅 / 🔊 읽기 / 🎮 화면 / 📊 프레젠테이션)
- 방송 정보 공유 기구 추가 (방송 URL과 썸네일을 플러그인 간 공유)
- VCI OSC 플러그인의 UTF-8 문자 깨짐 수정
- 커닝페이퍼 및 이미지 주입 UI 간소화
- 마작패 이미지 깜빡임 문제 수정
- AI가 각국의 표준 시간을 올바르게 인식하도록 개선 (일본: JST / 한국: KST / 러시아: MSK)
- 문서 전면 개편 (4개 언어)
