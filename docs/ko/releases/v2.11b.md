# 🚀 Release V2.11b：텔롭 세로 스택·줌·앵커 이동 지원!

> 👉 **[TeloPon v2.11b 다운로드](https://github.com/miyumiyu/TeloPon/releases/download/v2.11b/TeloPon-211b.zip)**

이번 업데이트는 **OBS 상의 텔롭 표시를 더욱 유연하게 제어할 수 있도록** 세로 스택·줌·앵커 이동 등 표시 UI 기능을 대폭 강화했습니다.
(🙏 Special Thanks **[@mo9mo9ke6ri](https://x.com/mo9mo9ke6ri)**)

---

## ✨ 주요 추가 및 개선 사항

### 🆕 텔롭 세로 스택

여러 텔롭이 겹치지 않도록 자동으로 세로로 쌓아서 표시됩니다.

* 텔롭은 **`window-reply` 그룹**（왼쪽）과 **일반 그룹**（오른쪽）으로 분리되어 독립 관리됩니다
* 각 그룹 최대 **4개**까지 동시 표시
* 새 텔롭이 위（또는 아래）에 나타나고, 기존 텔롭은 슬라이드하여 이동합니다
* 오래된 텔롭은 점차 투명해집니다（100% → 50% → 30% → 10%）

---

### 🆕 앵커 이동（드래그로 표시 위치 변경）

텔롭을 드래그하면 이후 텔롭이 해당 위치를 기준으로 나타납니다.

* **앵커 위치는 브라우저 `localStorage`에 저장**되어 OBS 재시작 후에도 유지됩니다
* 앵커 Y 좌표가 화면 하단 3/4 이하인 경우, 스택 방향이 **자동으로 위쪽으로 전환**됩니다

---

### 🆕 휠 줌으로 텔롭 크기 변경

텔롭에 마우스 커서를 올리고 휠을 조작하면 텔롭 크기를 변경할 수 있습니다.

* 줌 배율은 **`window-reply` 그룹**과 **일반 그룹**에 각각 독립적으로 저장됩니다
* 다음에 같은 그룹의 텔롭이 나타날 때 저장된 크기로 표시됩니다
* **더블클릭으로 크기를 초기화**할 수 있습니다（위치도 초기화）
* OBS 브라우저 소스의「**인터랙티브**」설정을 활성화해야 합니다
