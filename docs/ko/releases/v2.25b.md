## 🔧 수정 및 개선

### 🔄 리로드 기능 안정화
V2.24b에서 추가한 "리로드 버튼"의 안정성을 대폭 향상시켰습니다.

- **"멈춤" 현상 수정**: 리로드 후 AI가 응답하지 않게 되는 문제를 해결 (`recv_response`의 데드락 해결)
- **AI 상태 플래그 리셋 강화**: 새 세션 시작 시 `is_thinking` / `is_ai_generating` 등 내부 상태를 확실히 리셋
- **이상 상태 퍼지로부터의 복구 시간 단축**

리로드 버튼을 누른 후의 동작이 더욱 확실하고 매끄러워졌습니다.
