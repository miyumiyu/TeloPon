## 🆕 새로운 기능

### 🕐 정확한 시간 응답 (Function Calling)
"지금 몇 시야?" "오늘 며칠이야?"라고 물으면 AI가 시스템 시계를 조회하여 정확한 현지 시간으로 답합니다.  
Gemini Live API의 Function Calling을 활용하여 추측이나 기억이 아닌 실시간 시각 데이터를 참조합니다.

### 📺 썸네일·설명 AI 주입 ON/OFF
YouTube Live·YouTube Live+·Twitch·니코니코 각 플러그인에 방송 시작 시 AI에 보내는 썸네일 이미지와 설명의 ON/OFF 체크박스를 추가했습니다.  
불필요한 경우 OFF로 설정하여 프롬프트 토큰을 절약할 수 있습니다.

### 💬 텔롭을 Twitch 채팅에 게시
Twitch 플러그인 기능 설정에 "텔롭을 채팅에도 게시" 체크박스를 추가했습니다.  
ON으로 설정하면 AI가 출력하는 텔롭 내용이 Twitch 채팅에도 자동으로 게시됩니다.
