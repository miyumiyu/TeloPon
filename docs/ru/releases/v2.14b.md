# 🚀 Release V2.14b: Развязка плагинов, улучшение видимости телопов и VOICEVOX TTS!

> 👉 **[Скачать TeloPon v2.14b](https://github.com/miyumiyu/TeloPon/releases/download/v2.14b/TeloPon-214b.zip)**

В этом обновлении реализована развязка архитектуры плагинов, улучшена видимость телопов и добавлен плагин озвучки VOICEVOX.

---

## ✨ Развязка архитектуры плагинов

### 🆕 Принцип fire-and-forget

Хук вывода телопов (`on_telop_output`) переработан по принципу **fire-and-forget (выстрелил и забыл)**.

* Core **никогда не ждёт** завершения обработки плагином
* Ошибки или задержки в плагинах **не влияют на систему**
* Управление задержкой выделено в `set_telop_delay()` / `get_telop_delay()`

Это обеспечивает безопасную основу для работы сторонних плагинов.

👉 **[Руководство по разработке плагинов (подробности)](https://github.com/miyumiyu/TeloPon/blob/main/docs/ru/03_plugin_dev.md)**

---

## ✨ Улучшение видимости телопов

Прозрачность стека обычных телопов изменена для улучшения видимости старых телопов.

| Позиция (новейший → старый) | До | После |
|---|---|---|
| 1-й (новейший) | 100% | 100% |
| 2-й | 50% | **75%** |
| 3-й | 30% | **50%** |
| 4-й и далее | 10% | **25%** |

## ✨ Плагин озвучки VOICEVOX

### 🆕 Плагин VOICEVOX TTS

Новый плагин-расширение для озвучки текста телопов ИИ в реальном времени с помощью движка синтеза речи **VOICEVOX**.

* **Выбор персонажа и стиля**: все персонажи VOICEVOX + стили (Нормальный/Ласковый/Цундере и т.д.) через выпадающий список
* **Отображение иконки**: иконка выбранного персонажа/стиля в панели настроек
* **5 параметров голоса**: скорость, громкость, высота тона, интонация и тишина перед речью с мгновенной настройкой
* **Чтение TOPIC**: опционально читает верхнюю строку телопа вместе с основным текстом
* **Управление отображением телопа**: режим только озвучки (телоп скрыт) или отложенное отображение
* **Автоматическая проверка подключения**: автоматически проверяет соединение с VOICEVOX; UI неактивен при отсутствии связи

👉 **[Руководство по плагину VOICEVOX TTS](https://github.com/miyumiyu/TeloPon/blob/main/docs/ru/plugins/voicevox_plugin.md)**

> 💡 Требуется отдельная установка [VOICEVOX](https://voicevox.hiroshiba.jp/).
