## 🆕 Новые функции

### 🕐 Точные ответы о времени (Function Calling)
Спросите «Который час?» или «Какое сегодня число?» — ИИ обратится к системным часам и ответит точным местным временем.  
Используется Function Calling Gemini Live API — никаких догадок и галлюцинаций.

### 📺 Вкл/Выкл отправки миниатюры и описания ИИ
В плагинах YouTube Live, YouTube Live+, Twitch и Niconico добавлены чекбоксы для переключения отправки миниатюры и описания трансляции ИИ при старте эфира.  
Отключите, чтобы сэкономить токены промпта.

### 💬 Публикация телопов в чат Twitch
В настройках функций плагина Twitch добавлен чекбокс «Публиковать телопы в чат».  
При включении текст телопов ИИ автоматически публикуется в чат Twitch.
