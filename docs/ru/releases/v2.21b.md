## 🌟 Обзор

Полная поддержка API стриминговых платформ. Создание опросов, написание комментариев,
изменение названия трансляции, получение количества зрителей на YouTube Live, Twitch и Niconico Live —
**всё это теперь доступно простым голосовым обращением к ИИ**.

Встроенный менеджер плагинов позволяет добавлять нужные функции в один клик.
Плагины-расширения вынесены в отдельный репозиторий: от отображения превью и записи через OBS
до публикации твитов — управляйте каждым аспектом трансляции одним лишь голосом.

---

## 🆕 Новые возможности

### 📺 Полная поддержка API стриминговых платформ
- **YouTube Live**: Чтение/отправка комментариев, создание/подсчёт опросов, изменение названия трансляции, получение числа зрителей, поддержка Super Chat (OAuth)
- **Twitch**: Чтение/отправка комментариев, опросы, предсказания (ставки), создание клипов, изменение названия, получение числа зрителей (OAuth2 Device Code Flow)
- **Niconico Live**: Получение комментариев, уведомления о подарках/рекламе, создание/подсчёт опросов, публикация комментариев оператора, получение числа зрителей (повышен до встроенного плагина)

### 🔧 Менеджер плагинов
- Включение/отключение плагинов через интерфейс
- Загрузка и мгновенное использование плагинов-расширений в один клик
- Поддержка ручной установки пользовательских плагинов

### 🎤 Голосовое управление превью
- Скажите «покажи превью», «увеличь» или «убери», чтобы отобразить/управлять превью в OBS
- Автоматически активируется при подключении к стриминговой платформе

### 🎮 Улучшенная интеграция с OBS
- Голосовое управление записью OBS («начни запись», «останови запись»)
- Выбор источника захвата из выпадающего списка сцен

### 🗣️ Системные голосовые команды
- Скажите «закрой описание» или «убери табличку», и ИИ закроет окна на экране

## 📦 Плагины-расширения

Плагины-расширения теперь управляются через отдельный репозиторий.  
Их можно загрузить в один клик через менеджер плагинов.

🔗 **[TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)**

| Плагин | Описание |
|---|---|
| 📺 **YouTube Live+** | Интеграция YouTube через OAuth (комментарии, опросы, изменение названия, Super Chat) |
| 🐦 **X (Twitter)** | Получение хештегов, публикация твитов с превью/скриншотами |
| 🔊 **Озвучка субтитров (Windows)** | Автоматическое чтение встроенным голосом Windows |
| 🔊 **Озвучка субтитров (VOICEVOX)** | Синтез речи VOICEVOX |
| 📊 **Управление PowerPoint** | Голосовое управление слайд-шоу, автоинъекция заметок |
| 💬 **Discord** | Получение комментариев из каналов Discord |
| 💬 **Slack** | Получение комментариев из каналов Slack |
| 🎮 **VCI отправка субтитров** | VR-субтитры VirtualCast |

## 🔄 Улучшения и изменения

- Единая система именования плагинов (📺 Трансляции / 🐦 Соцсети / 💬 Чаты / 🔊 Озвучка / 🎮 Экран / 📊 Презентации)
- Добавлен механизм обмена информацией о трансляции (URL и превью доступны всем плагинам)

- Исправлено мерцание изображений маджонг-тайлов
- Улучшено распознавание ИИ региональных часовых поясов (Япония: JST / Корея: KST / Россия: MSK)
