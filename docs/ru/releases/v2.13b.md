# 🚀 Release V2.13b: Хук вывода телопов для плагинов и плагин TelopViewer!

> 👉 **[Скачать TeloPon v2.13b](https://github.com/miyumiyu/TeloPon/releases/download/v2.13b/TeloPon-213b.zip)**

В этом обновлении в систему плагинов добавлен хук вывода телопов, позволяющий плагинам получать вывод ИИ в реальном времени. Встроенный монитор телопов заменён плагинной реализацией.

---

## ✨ Хук вывода телопов для плагинов

### 🆕 Хук on_telop_output

Новый хук `on_telop_output` позволяет плагинам получать данные при каждом выводе телопа ИИ.

* Плагины получают полные данные: **topic / main / window / layout / badge**
* Теги `<b1>` `<b2>` и плитки маджонга (`<P1>` и т.д.) передаются как необработанные данные
* Возвращаемое значение управляет отображением телопов в OBS

| Значение | Поведение |
|---|---|
| `0` | Мгновенное отображение (по умолчанию) |
| `1+` | Отображение через N секунд (задержка) |
| `-1` | Подавление отображения в OBS (плагин всё равно получает данные) |

### 🛡️ Механизмы безопасности

* `on_telop_output` каждого плагина выполняется с **таймаутом 1 секунда**
* После **3 таймаутов** плагин автоматически исключается из вызовов хука

👉 **[Руководство по разработке плагинов (подробности on_telop_output)](https://github.com/miyumiyu/TeloPon/blob/main/docs/ru/03_plugin_dev.md)**

## ✨ Монитор телопов заменён плагином TelopViewer

### 🆕 Плагин TelopViewer

Встроенный «Монитор телопов» удалён и перестроен как плагин **TelopViewer** с использованием хука `on_telop_output`.

* **Отображение в реальном времени** вывода телопов ИИ в панели управления
* **Цветовая маркировка тегов B1/B2** (B1=красный, B2=голубой)
* **Настройка задержки отображения через выпадающий список** (0/1/2/3/5/10/-1)
* Просто откройте из кнопки «Панель управления» в списке плагинов
