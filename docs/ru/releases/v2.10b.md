# 🚀 Release V2.10b: Отображение плиток маджонга, аватары зрителей и улучшение стабильности!

> 👉 **[Скачать TeloPon v2.10b](https://github.com/miyumiyu/TeloPon/releases/download/v2.10b/TeloPon-210b.zip)**

В этом обновлении основное внимание уделено **дальнейшей стабилизации и оптимизации Gemini 3.1 Flash Live**, а также добавлена поддержка трансляций маджонга, отображение аватаров зрителей и многое другое.

---

## ✨ Новое и улучшенное

### ⚡ Оптимизация и стабилизация Gemini 3.1 Flash Live

Обработка подключения для Gemini 3.1 Flash Live была полностью переработана.

* Скорость отображения телопов возросла, обеспечивая почти реальное время отображения
* Улучшена стабильность переподключения сессий

---

### 🆕 Улучшение ответов на комментарии (window-reply)

Телоп ответа на комментарии `window-reply` значительно улучшен по дизайну и функциональности.

* Изменён на стиль **облачка речи**, размещённого в **левом верхнем углу** экрана
* Отображает **аватары (миниатюры)** авторов комментариев **YouTube и Twitch** в телопах ответов
  * Для YouTube аватары загружаются автоматически при получении комментариев
  * Для Twitch используется Helix API (только при настроенных Client ID и Secret ID) → [Настройка плагина Twitch](../plugins/TwitchPlugin.md)

---

### 🆕 Поддержка трансляций маджонга

Добавлены новые функции для трансляций маджонга.

#### Встроенные теги отображения плиток маджонга

Если написать теги плиток в промпте, изображения плиток автоматически отображаются в телопе.

| Тип | Теги |
|---|---|
| Символы (Ман) | `<M1>` 〜 `<M9>` |
| Бамбук (Со) | `<S1>` 〜 `<S9>` |
| Круги (Пин) | `<P1>` 〜 `<P9>` |
| Почётные | `<J1>`Восток `<J2>`Юг `<J3>`Запад `<J4>`Север `<J5>`Белый `<J6>`Зелёный `<J7>`Красный |

#### Промпт советника по маджонгу `r_mahjong.txt`

* В сочетании с захватом OBS ИИ может отображать советы по сбросу плиток
* Разработан для предоставления советов только на основе плиток, видимых на экране
* ⚠️ **ИИ может неправильно прочитать изображение и упомянуть несуществующие плитки.** Воспринимайте это как развлечение!
