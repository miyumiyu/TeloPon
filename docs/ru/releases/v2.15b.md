# 🚀 Release V2.15b: Иконка ИИ, обновление интерфейса и повышение стабильности!

> 👉 **[Скачать TeloPon v2.15b](https://github.com/miyumiyu/TeloPon/releases/download/v2.15b/TeloPon-215b.zip)**

В этом обновлении добавлено отображение иконки ИИ на телопах, реорганизован интерфейс настроек и повышена стабильность соединения.

---

## ✨ Иконка ИИ

### 🆕 Отображение иконки ИИ на телопах

Теперь можно отображать пользовательскую иконку в левой части телопа, соответствующую характеру ИИ.

* Установите изображение через кнопку «Иконка ИИ» в интерфейсе
* **Автоматическое изменение размера под высоту рамки** (квадрат, обрезка по центру)
* Без иконки — отображение как раньше
* Поддерживается: variety / news / digital / cute / simple
* Не отображается на reply / explain / nameplate

---

## ✨ Обновление интерфейса

Макет настроек переработан для большей наглядности.

* **⚙️ Настройки ИИ**: API Key, статус авторизации, модель, уровень мышления, личность ИИ, иконка ИИ
* **📡 Настройки трансляции**: имя стримера, микрофон, индикатор, порог, тема

---

## 🔧 Повышение стабильности

### Автоматическое восстановление при отсутствии ответа ИИ

Добавлен механизм автоматического переподключения сессии при длительном отсутствии ответа от ИИ.
Значительно снижена частота ситуаций, когда ИИ замолкает во время трансляции.

---

## 🔧 Прочие исправления

* Исправлена проблема, из-за которой некоторые плагины перестали работать в v2.14b (добавлен ctypes.wintypes в сборку)
* Исправлена проблема сохранения задержки телопа при перезапуске (теперь всегда сбрасывается в 0)
