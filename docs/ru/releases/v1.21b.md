# 🎯 Release V1.21b: Улучшена стабильность ответов ИИ и отображение статуса!

Это обновление сосредоточено на **качестве ответов ИИ** и **обратной связи в реальном времени**.
Оптимизация параметров режима обдумывания снижает количество пропущенных ответов, а улучшенное отображение статуса позволяет с первого взгляда понять, что делает ИИ.
Корректировки промптов также помогают ИИ более естественно брать инициативу в разговоре.

---

## ✨ Улучшения и исправления

### 🧠 Оптимизация параметров ИИ для повышения частоты ответов

Значения по умолчанию параметров, связанных с режимом обдумывания, были скорректированы.

* **temperature**: 0.7 → **1.0**
* **top_p**: 0.8 → **0.95**
* **thinking_budget**: 2048 → **1024**

Снижение thinking_budget предотвращает чрезмерное обдумывание и уменьшает количество «пустых ходов», когда ответ не возвращается.
Повышение temperature и top_p расширяет выразительный диапазон ИИ для более естественных ответов.

---

### ⏱️ Улучшено отображение статуса «Обработка»

Раньше, когда ИИ находился в состоянии «обработки», не было возможности понять, когда оно завершится.

* Статус теперь отображает **обратный отсчёт** оставшихся секунд, давая чёткое представление о времени ожидания
* По истечении таймаута (10 секунд) состояние обработки автоматически сбрасывается, и ИИ готов к следующему вводу

---

### 💬 Улучшения промптов (日本語 / English / 한국어 / Русский)

Во все стандартные промпты на всех языках добавлены инструкции, побуждающие ИИ активно высказываться.

* ИИ теперь может естественно развивать темы, не ограничиваясь прошлым ходом разговора
* Вместо молчания во время пауз ИИ будет проявлять инициативу и говорить самостоятельно

---

## 💡 Советы для стримеров

**Замечаете пропущенные ответы?**
Это обновление напрямую решает данную проблему. Снижение thinking_budget в частности уменьшает паттерн «слишком много думает, ничего не говорит».
Если проблема сохраняется, попробуйте опцию `-tb 0` для полного отключения режима обдумывания.
