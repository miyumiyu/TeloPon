# 🚀 Release V2.12b: Полное обновление UI/UX телопов и настройка уровня мышления!

> 👉 **[Скачать TeloPon v2.12b](https://github.com/miyumiyu/TeloPon/releases/download/v2.12b/TeloPon-212b.zip)**

В этом обновлении полностью переработаны визуальное оформление и взаимодействие с телопами. Также добавлена поддержка настройки уровня мышления Gemini 3.1.

---

## ✨ Полное обновление UI/UX телопов

### 🆕 Дизайн карточки SNS для window-reply

Телопы `window-reply` переработаны в стиле карточек-ответов в социальных сетях.

* **Круглый аватар** автора комментария (поддерживаются изображения профиля YouTube и др.)
* **Значок ↩** в правом нижнем углу аватара
* **Анимация появления слайдом** (направление определяется автоматически по позиции якоря)
* Тёмный фон с **эффектом свечения** (поддерживаются все 4 темы: default / comic / honwaka / tsundere)

---

### 🆕 Плавность анимаций

Полностью улучшены анимации перемещения, появления и исчезновения телопов.

* Появление: **плавное увеличение от scale(0)**
* Исчезновение: **плавное уменьшение до scale(0)** (исправлено резкое исчезновение на половинном размере)
* Перемещение: **мгновенная реакция без задержек и подёргиваний**
* Переключение очереди: при наличии следующего элемента **затухание и появление перекрываются** для бесшовного перехода

---

### 🆕 Кнопка «Далее» для explain / Кнопка «×» для nameplate

Новые кнопки для удобного управления телопами.

* **Рамка explain**: при наличии элементов в очереди показывает количество + значок «›» в правом нижнем углу. Клик для мгновенного перехода к следующему элементу
* **Рамка nameplate**: постоянно отображаемая кнопка «✕» в правом верхнем углу. Клик для мгновенного закрытия

---

### 🆕 Улучшение перетаскивания explain

Значительные улучшения взаимодействия с `window-explain` (окном объяснения).

* Позиция после перетаскивания **сохраняется после перезапуска OBS**
* Масштаб после прокрутки колёсиком **также сохраняется**
* **Двойной клик сбрасывает** позицию и масштаб
* При упоре в край вьюпорта рамка **сжимается горизонтально**, при возврате — расширяется (фиксированный отступ 40px)
* **Таймер приостанавливается во время перетаскивания** (телоп не исчезнет, пока вы его перемещаете)

---

## ✨ Настройка уровня мышления

### 🆕 Поддержка уровня мышления Gemini 3.1

Добавлена поддержка новой функции мышления Gemini 3.1. Настройка через выпадающий список «💭 Уровень мышления» в интерфейсе.

| Уровень | Описание |
|---|---|
| **minimal** | Минимальная задержка (по умолчанию) |
| **low** | Для простых задач |
| **medium** | Для средней сложности |
| **high** | Для сложных рассуждений и многоэтапного планирования |

* Устаревший параметр `thinking_budget` (количество токенов) удалён
* Настройки сохраняются в `settings.json` и восстанавливаются при следующем запуске
