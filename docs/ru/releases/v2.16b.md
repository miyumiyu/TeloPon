# TeloPon V2.16b — Примечания к выпуску

## 🎮 Масштабное обновление плагина Twitch

С поддержкой OAuth2-аутентификации интеграция с Twitch полностью обновлена — от простого чтения чата до полнофункциональной работы.

- **Запись комментариев** — «Напиши в чат "привет"», и ИИ публикует в чат Twitch
- **Опросы** — «Создай опрос о любимых цветах» для создания голосования, «Подведи итоги» для объявления результатов
- **Предсказания (ставки)** — «Пусть зрители сделают ставку, смогу ли я победить босса» для ставок баллами канала (Филиал/Партнёр)
- **Создание клипов** — «Это было круто! Сделай клип!» для сохранения последних 30 секунд
- **Изменение заголовка** — «Измени заголовок на "Общение"» для мгновенного обновления
- **Число зрителей** — «Сколько зрителей?», и ИИ сообщит
- **Исключение ботов** — Двойной клик для добавления nightbot и др. в список исключений
- **Режим авторизации / IRC** — Переключение между полным функционалом и только чтением чата (без авторизации)

## ▶️ Плагин YouTube OAuth добавлен в расширения

Выпущен продвинутый плагин интеграции с YouTube на базе YouTube Data API + OAuth2.

- **Запись комментариев** — «Напиши комментарий "привет"», и ИИ публикует в чат YouTube
- **Опросы** — «Создай опрос» для создания официального опроса YouTube, «Подведи итоги» для объявления результатов
- **Изменение заголовка** — «Измени заголовок на ○○» для обновления названия трансляции
- **Число зрителей** — «Сколько зрителей?» для получения числа одновременных зрителей
- **Выбор трансляции** — Выбор из списка запланированных трансляций или указание по URL
- **Автоопределение прав** — Автоматическое определение владельца/менеджера и отображение доступных функций

Также подготовлено руководство по настройке GCP со скриншотами.

👉 [TeloPon Extensions](https://github.com/miyumiyu/TeloPon-Extensions)

## 🛡️ Права на использование команд в Twitch / YouTube OAuth

В каждую функцию плагинов Twitch и YouTube OAuth добавлен флажок «Зритель».  
Управляйте тем, могут ли зрители запускать команды через комментарии — для каждой функции отдельно.  
Например, изменение заголовка и опросы можно ограничить только голосовыми командами стримера.

## 🎤 Закрытие окон объяснения и таблички голосом

Скажите «Закрой объяснение» или «Скрой табличку», и ИИ закроет окно на экране.

## 🐛 Исправления ошибок

- Исправлено мерцание изображений плиток маджонга
- ИИ теперь правильно определяет местное время (Япония: JST / Корея: KST / Россия: MSK)

## 📦 Обновление структуры плагинов

Плагины-расширения перенесены в отдельный репозиторий [TeloPon-Extensions](https://github.com/miyumiyu/TeloPon-Extensions).  
Плагины, промпты и темы теперь управляются в единой новой структуре.
