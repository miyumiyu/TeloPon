Это первый пакет официальных расширяющих плагинов, которые сделают TeloPon ещё удобнее!
Просто скачайте нужный файл (`.py`) и добавьте его в свой TeloPon.
**※ Доступно начиная с версии v1.05b.**

## 📦 Содержимое пакета

### 💬 Интеграция с Discord в реальном времени (`discord_integration.py`)
Получает комментарии из указанного канала Discord-сервера в реальном времени и передаёт их ИИ.
* **Сверхпростая настройка**: просто введите Bot-токен и нажмите кнопку — URL для приглашения генерируется автоматически.
* **Полностью в реальном времени**: комментарии подхватываются мгновенно, без задержек.

### 🏢 Интеграция с комментариями Slack (`slack_integration.py`)
Получает комментарии из указанного канала Slack-воркспейса в реальном времени и передаёт их ИИ.
* **Поддержка Socket Mode**: использует современный метод связи, применяемый в корпоративной среде — приём полностью в реальном времени без задержек.
* **Автоматическое преобразование имён**: внутренние ID пользователей Slack автоматически преобразуются в отображаемые имена перед озвучиванием ИИ.

## 🛠️ Как установить плагин

1. Прокрутите страницу вниз до раздела **Assets** и нажмите на нужный файл `.py`, чтобы скачать его.
2. Откройте папку `TeloPon-XXX` на вашем компьютере.
3. Перетащите скачанный файл `.py` в папку **`plugins`** внутри неё.
4. Запустите TeloPon — новая интеграция появится в списке «🔌 Расширения» справа!

※ Подробные инструкции по настройке смотрите в экране настроек (⚙️) каждого плагина или в официальной документации.
