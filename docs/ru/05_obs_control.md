# 🎮 Как управлять телопами в OBS

Телопы TeloPon можно свободно **перетаскивать** и **масштабировать** прямо в OBS.
На этой странице описано, как управлять телопами в браузерном источнике OBS.

---

## 1. Войдите в режим «Взаимодействие» OBS

Нажмите кнопку **«Взаимодействие» (Interact)** внизу области предпросмотра OBS.

![Кнопка взаимодействия OBS](../../images/obsctl.png)

> После нажатия область предпросмотра будет обведена красной рамкой, что означает активацию взаимодействия с браузерным источником.

---

## 2. Управление телопами

В режиме взаимодействия вы можете управлять телопами напрямую с помощью мыши.

![Режим взаимодействия OBS](../../images/obsctl2.png)

### Перетаскивание для перемещения

| Объект | Действие |
|---|---|
| **Обычные телопы** (variety / news / reply и др.) | Перетащите в любую позицию. Позиция сохраняется после перезапуска OBS |
| **Рамка объяснения** (explain) | Перетащите для перемещения. Позиция сохраняется после перезапуска OBS |
| **Табличка имени** (nameplate) | Отображается внизу экрана. Нажмите «✕» в правом верхнем углу, чтобы закрыть |

### Колёсико мыши для масштабирования

Прокрутите **колёсико мыши** над телопом для увеличения или уменьшения.

* **Колёсико вверх** → Увеличение
* **Колёсико вниз** → Уменьшение
* Масштаб сохраняется для каждой группы

### Двойной клик для сброса

**Дважды кликните** по телопу, чтобы сбросить его позицию и масштаб в исходное состояние.

### Долгое нажатие для удаления

**Нажмите и удерживайте** телоп в течение 1 секунды, чтобы вручную его убрать.

---

## 3. Специальные функции рамки объяснения (explain)

Рамка объяснения имеет несколько дополнительных функций.

| Действие | Результат |
|---|---|
| **Перетаскивание** | Свободное перемещение (позиция сохраняется после перезапуска OBS) |
| **Колёсико** | Увеличение/уменьшение (масштаб тоже сохраняется) |
| **Двойной клик** | Сброс позиции и масштаба |
| **Долгое нажатие (1 с)** | Удалить текущее объяснение и перейти к следующему |
| **Перетаскивание к краю** | Рамка сжимается горизонтально при упоре в край экрана. Возвращается при оттаскивании |
| **Значок «Далее»** | При наличии элементов в очереди в правом нижнем углу появляется значок. Нажмите для перехода к следующему элементу |
