# 🔑 Руководство для начинающих: Получите бесплатный API-ключ!

Для запуска TeloPon вам нужен AI-ключ (API-ключ), предоставляемый Google.
Доступен **полностью бесплатный тарифный план без необходимости кредитной карты**, которого вполне достаточно для личных стримов и хобби!

Даже если вы новичок, вы легко можете получить его с ПК или смартфона — просто следуйте инструкциям ниже по порядку.

---

### Шаг 1: Откройте страницу создания API-ключа
Сначала перейдите в Google AI Studio, войдите в систему и нажмите кнопку **"Create API key"** на экране.

👉 **[Google AI Studio (страница API-ключей)](https://aistudio.google.com/api-keys)**

<img src="../../images/api-key1.png" width="300">

### Шаг 2: Создайте новый проект
Нажмите «Select an imported project», затем выберите **«+ Create project»** из меню.

<img src="../../images/api-key2.png" width="300">

### Шаг 3: Назовите проект
Появится экран «Create new project». Введите описательное имя в поле «Name your project» (например, **`TeloPon`**), затем нажмите «Create project».

<img src="../../images/api-key3.png" width="300">

### Шаг 4: Назовите API-ключ
Далее появится экран «Create a new key». Введите имя в поле «Key name» (например, **`TeloPon API Key`**), затем нажмите «Create key».

<img src="../../images/api-key4.png" width="300">

### Шаг 5: Скопируйте API-ключ
После создания вы вернётесь к списку API-ключей. Ваш новый ключ будет показан (например, `...3eb8`). Нажмите на него, чтобы скопировать полную строку (начинающуюся с `AIza...`).

<img src="../../images/api-key5.png" width="300">

---

> ⚠️ **Важное предупреждение (обращайтесь осторожно!)**
> **Никогда не делитесь этим API-ключом с кем-либо и не допускайте его появления на вашем стриме.**
> Относитесь к нему как к ключу от дома или паролю — вставьте его в надёжное место, например в блокнот. Он понадобится вам при следующей настройке приложения!
