# 🏢 Интеграция Slack (slack_integration.py)

С этим плагином ИИ TeloPon зачитывает комментарии, размещённые в вашем корпоративном Slack-рабочем пространстве или командных каналах, с нулевой задержкой!
Он использует новейшую технологию «Socket Mode», обеспечивая удивительно быстрые реакции в реальном времени.

Настройка состоит из трёх основных шагов: **«① Установка плагина»**, **«② Создание Slack-бота (получение двух токенов)»** и **«③ Настройка в TeloPon»**. Шагов довольно много, но если следовать им по порядку, вы обязательно справитесь!

---

## 📥 Шаг 1: Установите плагин и подготовьтесь

Сначала добавьте расширение в вашу установку TeloPon.

1. Скачайте **`slack_integration.py`** с
   🔗 [TeloPon Official Extension Plugin Pack v1.0 (Discord & Slack)](https://github.com/miyumiyu/TeloPon/releases/tag/plugins-v1.0)
2. Поместите скачанный файл `slack_integration.py` в папку **`plugins`** внутри вашей главной папки TeloPon.
3. Запустите TeloPon — если **«🏢 Интеграция Slack»** появится в списке «🔌 Расширения» на главном экране, готово!

---

## 🛠️ Шаг 2: Создайте Slack-бота и получите «Два токена»

Чтобы ИИ мог читать комментарии Slack, создайте специального «Бота (приложение)». Здесь вы получите **2 типа токенов (ключей)**.

### 1. Создайте приложение и дайте ему имя
1. В браузере ПК перейдите на [Slack API (Your Apps)](https://api.slack.com/apps) и нажмите **«Create New App»** в верхнем правом углу.
2. Выберите **«From scratch»**, введите понравившееся имя (например, `TeloPon Bot`), выберите рабочее пространство для установки и нажмите **«Create App»**.
3. **[Важно]** Откройте **«App Home»** в левом меню, прокрутите вниз до **«Your App's Presence in Slack»** и нажмите кнопку «Edit», чтобы задать `Display Name` (отображаемое имя) и `Default Username` (внутренний ID, только строчные буквы и цифры), затем сохраните. Без этого шага вы не сможете пригласить бота!

### 2. Включите Socket Mode и получите «App-Level Token»!
1. Откройте **«Socket Mode»** из левого меню.
2. Переведите переключатель **«Enable Socket Mode»** в положение **ON (зелёный)**.
3. Когда запросят имя токена, введите что угодно (например, `telopon-socket`) и нажмите «Generate».
4. Появится строка, начинающаяся с **`xapp-`**. Это **[App-Level Token]**. Скопируйте и запишите его.

### 3. Выдайте разрешения на чтение сообщений (Scopes)
1. Откройте **«OAuth & Permissions»** из левого меню.
2. Прокрутите вниз, и в разделе **«Bot Token Scopes»** секции **«Scopes»** нажмите «Add an OAuth Scope» и добавьте следующие 5 пунктов:
   * `channels:history` / `groups:history` / `channels:read` / `groups:read` / `users:read`

### 4. Настройте получение в реальном времени (Event Subscriptions)
1. Откройте **«Event Subscriptions»** из левого меню.
2. Включите **«Enable Events»** вверху (**ON (зелёный)**).
3. Откройте **«Subscribe to bot events»** чуть ниже, добавьте следующие 2 события через «Add Bot User Event» и нажмите **«Save Changes»** в нижнем правом углу:
   * `message.channels` / `message.groups`

### 5. Установите в рабочее пространство и получите «Bot Token»!
1. Откройте **«Install App»** из левого меню и нажмите **«Install to Workspace»**. (※ Если разрешения изменились, будет написано «Reinstall».)
2. Появится экран авторизации — нажмите «Allow».
3. Появится строка, начинающаяся с **`xoxb-`**. Это **[Bot Token]**. Скопируйте и запишите его.

### 6. [КРИТИЧНО] «Пригласите» бота в канал Slack
В Slack **упомяните бота в канале, который хотите отслеживать** (например, введите `@TeloPon Bot` в `#general`). Когда спросят «Добавить в канал?», обязательно добавьте. Без этого шага он не сможет читать комментарии!

---

## 🖥️ Шаг 3: Настройте и подключитесь в TeloPon

Вернитесь на экран TeloPon и откройте панель настроек (⚙️) для «🏢 Интеграция Slack».

![Панель настроек интеграции Slack](../../../images/discord_integration.png)

1. **Введите токены**
   * Вставьте строку **`xapp-`**, полученную на шаге 2, в поле «App-Level Token».
   * Вставьте строку **`xoxb-`**, полученную на шаге 2, в поле «Bot Token».
2. **Получите список каналов**
   * Нажмите кнопку **«🔄 Проверить токены и получить список каналов»**.
   * При успехе в списке появятся каналы вашего Slack-рабочего пространства.
3. **Выберите канал для мониторинга**
   * Выберите канал для мониторинга из выпадающего списка. (Публичные каналы отображаются с 💬, приватные — с 🔒.)
4. **Подключитесь!**
   * Наконец, нажмите синюю кнопку **«Подключить»** — вы готовы, когда статус становится зелёным: «⚡ Статус: Подключено (Socket Mode активен)»!

## 💡 Советы и примечания по использованию

После настройки просто нажмите «Начать прямое подключение (Запустить ИИ)» с главного экрана TeloPon! ИИ будет быстро реагировать на сообщения в указанном канале Slack.

* **Автоматическое преобразование имён**: Уникальные ID пользователей Slack (U1234... и т.д.) автоматически преобразуются в отображаемое имя человека (например, Иван Петров) на бэкенде TeloPon перед отправкой ИИ. ИИ будет правильно обращаться к людям по имени в ответах!
* **Настройка промпта**: Изменение текста инструкций в нижней части панели настроек на что-то вроде «Это чаты от внутренних членов команды! Подними всем настроение!» может раскрыть ещё больше личности ИИ.
