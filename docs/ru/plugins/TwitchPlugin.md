# 🎮 Инструмент интеграции с Twitch (TwitchPlugin.py)

Плагин интеграции с Twitch.  
ИИ автоматически выполняет различные операции Twitch по голосовым командам стримера.

<img src="../../../images/TwitchPlugin.png" width="600">

---

## 🌟 Основные функции

| Функция | Описание | Ограничение |
|---|---|---|
| Чтение чата | Комментарии зрителей передаются ИИ | Нет |
| Запись в чат | ИИ публикует в чат Twitch | Нет |
| Опросы (Poll) | Создание и подсчёт голосов | **Только филиал/партнёр** |
| Предсказания | Зрители ставят баллы канала | **Только филиал/партнёр** |
| Изменение заголовка | Голосовая смена названия | Нет |
| Создание клипов | Клип последних ~30 секунд | Нет |
| Число зрителей | Текущее число зрителей | Нет |
| Исключение ботов | Исключение определённых пользователей | - |

## ⚙️ Руководство по настройке

### Шаг 1: Выбор режима подключения

#### Режим IRC (любой канал, только чтение чата)

**Аутентификация не требуется.**

> Пропустите шаги 2-3 и перейдите к шагу 4.

#### Режим авторизации (ваш канал, все функции)

**Требуется OAuth2-аутентификация.**

---

### Шаг 2: Регистрация приложения в Twitch Developer Console

1. Войдите в [Twitch Developer Console](https://dev.twitch.tv/console/apps)
2. Нажмите «Зарегистрировать приложение»
3. Скопируйте **Client ID**

4. **Client Secret (рекомендуется):**

   > ⚠️ Без Client Secret токен истекает через ~4 часа и потребуется повторная аутентификация.

### Шаг 3: Аутентификация в TeloPon

1. TeloPon → Расширения → «Twitch» → шестерёнка
2. Введите Client ID и Client Secret
3. Нажмите «Войти через Twitch»
4. Завершите аутентификацию в браузере

---

### Шаг 4: Подключение и начало трансляции

1. **Режим авторизации:** Нажмите «Подключить»
2. **Режим IRC:** Введите имя канала и нажмите «Подключить»
3. На главном экране TeloPon нажмите **«🔴 Начать трансляцию»**

> **Важно:** Нажмите «Подключить» **до** начала трансляции.

## 🎙️ Примеры голосовых команд

| Стример говорит | Действие ИИ |
|---|---|
| «Напиши в чат "привет"» | Публикует «привет» в чат |
| «Создай опрос о любимых цветах» | Создаёт опрос на 60 секунд |
| «Подведи итоги» | Закрывает опрос и объявляет результаты |
| «Пусть зрители сделают ставку» | Создаёт предсказание |
| «Правильный ответ — номер 1!» | Завершает предсказание |
| «Измени заголовок на "Общение"» | Меняет заголовок трансляции |
| «Сделай клип!» | Создаёт клип последних ~30 секунд |
| «Сколько зрителей?» | Показывает число зрителей |

---

## 👤 Управление пользователями (исключение ботов)

**Двойной клик** по недавнему пользователю добавляет в список исключений.  
**Двойной клик** по исключённому пользователю удаляет из списка.
