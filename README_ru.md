# TeloPon 🎙️✨

**ИИ нового поколения, который слушает голос стримера и в реальном времени создаёт субтитры, комментирует и помогает вести трансляцию**

![Демо TeloPon](images/demo_ru.gif)

## 🌟 Что умеет TeloPon

### 💬 ИИ реагирует в реальном времени
Голос стримера обрабатывается напрямую через Gemini Live API. Этапы «распознавание речи → преобразование в текст → отправка в ИИ» полностью исключены, что обеспечивает **сверхнизкую задержку ответов, не нарушающую темп разговора**.

### 📺 Интеграция со стриминговыми платформами
ИИ автоматически подхватывает и реагирует на комментарии YouTube Live, Twitch и Niconico Live. Миниатюра, название и описание трансляции также передаются ИИ, чтобы он понимал содержание программы и вместе с вами вёл эфир.

### 🎤 Голосовое управление
Просто говорите, и ИИ поможет с трансляцией. Создание опросов, изменение заголовка, создание клипов, запись в OBS, отображение миниатюры, публикация твитов — всё это можно делать **только голосом**.

### 🎮 Комментирование игрового экрана
Благодаря интеграции с OBS можно показать ИИ игровой экран или изображение с камеры. ИИ выдаёт реакции и комментарии в реальном времени на основе визуальной информации. Запуск/остановку записи OBS тоже можно контролировать голосом.

### 🐦 Интеграция с соцсетями
Автоматическое получение хэштегов X (Twitter) и передача их ИИ. ИИ автоматически публикует твиты с миниатюрой или скриншотом игры.

### 🔊 Озвучка субтитров
Автоматическая озвучка субтитров стандартным голосом Windows (SAPI5), VOICEVOX или Gemini 3.1 Flash TTS. Добавьте «голос» ИИ в вашу трансляцию.

### 📊 Поддержка презентаций
Голосовое управление слайд-шоу PowerPoint. Переключение слайдов вперёд/назад, переход к конкретному слайду, затемнение экрана. Заметки к слайдам автоматически передаются ИИ для поддержки презентации.

### 🧠 Свой уникальный ИИ-персонаж
Достаточно написать промпт (текстовый файл), чтобы создать ИИ с любым характером. Цундэрэ, с акцентом, язвительный… полная свобода.

### 🎨 Свободный дизайн субтитров
Полная настройка внешнего вида субтитров через CSS. Звуковые эффекты тоже настраиваются.

## 🎤 Список голосовых команд для ИИ

Просто скажите во время трансляции, и ИИ автоматически выполнит команду.

| Категория | Пример голосовой команды | Действие |
|---|---|---|
| **Миниатюра** | «Покажи миниатюру» | Отображение миниатюры в OBS |
| | «Увеличь/уменьши/убери миниатюру» | Увеличение, уменьшение, скрытие |
| **Запись OBS** | «Начни запись» / «Останови запись» | Управление записью OBS |
| **Интерфейс** | «Убери описание» / «Убери имя» | Скрытие окон субтитров |
| **YouTube** | «Создай опрос» / «Измени заголовок» | Операции на стороне YouTube |
| **Twitch** | «Создай клип» / «Сделай предсказание» | Операции на стороне Twitch |
| **Niconico** | «Напиши в операторский комментарий ○○» | Операции на стороне Niconico |
| **X (Twitter)** | «Опубликуй твит» / «Твитни с миниатюрой» | Публикация твитов |
| **PowerPoint** | «Следующий слайд» / «Начни презентацию» | Управление слайдами |

> Каждая команда доступна только при активированном соответствующем плагине.

## 🔑 Подготовка: Получение бесплатного API-ключа

Для работы TeloPon необходим API-ключ Google Gemini (**полностью бесплатно, без привязки карты**).

1. Войдите с помощью аккаунта Google в **[Google AI Studio](https://aistudio.google.com/)**
2. Нажмите «**Get API key**» → «**Create API key**»
3. Скопируйте строку, начинающуюся с `AIza...`

👉 **[Подробная инструкция с картинками](docs/ru/04_get_apikey.md)**

## 🎛️ Использование интерфейса (UI)

![Экран интерфейса](images/ui_ru.png)

### ⚙️ Настройки ИИ
| Параметр | Описание |
|---|---|
| 🔑 **API Key** | Вставьте API-ключ Gemini и нажмите «Аутентификация» |
| 🧠 **Модель** | Обычно оставьте `Auto` |
| 💭 **Уровень мышления** | От `minimal` (быстрый) до `high` (глубокий анализ). Для стримов рекомендуется `minimal` |
| 🧠 **Личность ИИ** | Выберите скрипт из папки `prompts/` |
| 🎥 **Имя стримера** | Имя, которым ИИ будет к вам обращаться (обязательно) |

### 🎤 Настройка микрофона
Выберите используемый микрофон. Оптимальная громкость — когда индикатор уровня **жёлтый**. Красный означает перегрузку.  
**Ползунком порога** настройте уровень так, чтобы фоновая музыка не подхватывалась.

### ⏱️ Настройки времени
| Параметр | Описание |
|---|---|
| **Обычное / Особое** | Время отображения субтитров (в секундах) |
| **Режим речи** | `None` / `Авто-речь` (предотвращение тишины) / `Авто-сегмент` (периодическое резюме) |
| **Время вмешательства** | Секунды ожидания до автоматического вмешательства |

### 🔌 Расширения (Плагины)
На правой панели расположены плагины. Настройка и управление — через «Панель управления» каждого плагина.

### 🔧 Менеджер плагинов
Кнопка **«🔧 Менеджер плагинов»** в верхней части списка плагинов.

| Вкладка | Возможности |
|---|---|
| **Активные** | Управление плагинами, отображаемыми в интерфейсе. Скрытие ненужных |
| **Отключённые** | Повторное включение скрытых плагинов |
| **Доступные для скачивания** | Скачивание и мгновенное использование расширенных плагинов в один клик |
| **Ручная установка** | Установка `.py` файлов собственных плагинов |

### 🎨 Тема и звуковые эффекты
Выбор дизайна субтитров через тему (CSS). Включение/выключение звуковых эффектов и регулировка громкости.

### 🔗 Копирование URL для OBS
1. Нажмите «🔗 Копировать OBS URL»
2. OBS Studio → Добавить источник «Браузер» → Вставить URL
3. Установите размер **ширина 1920 × высота 1080**

👉 **[Руководство по управлению субтитрами в OBS (с иллюстрациями)](docs/ru/05_obs_control.md)**

**👉 После завершения всех настроек нажмите «🔴 Начать прямое подключение»!**

## 💡 Советы для длительных трансляций

Примерно через 30 минут память ИИ заполняется и обработка замедляется.  
При смене темы просто нажмите **«⬛ Отключить» → «🔴 Начать прямое подключение»** — память ИИ сбрасывается, и он возвращается к быстрым и точным ответам.

## ❓ Если что-то не работает

| Симптом | На что обратить внимание |
|---|---|
| «Ошибка аутентификации» | Проверьте, нет ли лишних пробелов вокруг API-ключа |
| ИИ постоянно «пропускает» | Проверьте выбор микрофона и ползунок порога |
| В OBS ничего не отображается | Вставьте URL в источник «Браузер» заново |
| Не воспроизводятся звуковые эффекты | Снимите флажок «Управлять аудио через OBS» в свойствах источника браузера OBS |
